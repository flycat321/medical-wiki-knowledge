### 2.1 各方职责界面

| 角色 | 职责范围 | 与设计院的关系 |
|------|---------|--------------|
| 业主（甲方） | 项目决策、资金、需求确认、验收 | 设计院的最终服务对象 |
| EPC总包方 | 项目总体管理、施工组织、采购、质量安全 | 设计院的合同甲方（EPC模式下） |
| 设计分包（设计院） | 全过程设计、优化、配合 | 核心技术服务方 |
| 施工分包 | 各专业施工 | 设计院提供技术支持 |
| 监理 | 施工质量/进度/安全监督 | 设计院配合技术答疑 |
| 各专项分包 | 专项设计+施工 | 设计院审核管控 |

### 2.2 ★设计院在EPC中的核心角色

```
设计院在EPC项目中的角色定位：

  不只是"画图的" ──→ 而是"技术总控方"

  具体体现：
  1. 技术决策参与者 —— 参与材料/设备/方案的技术比选
  2. 造价控制推动者 —— 主导限额设计和设计优化
  3. 各方协调桥梁 —— 协调业主需求与施工可行性
  4. 质量控制把关者 —— 审核分包图纸、参与关键节点验收
  5. 变更管理执行者 —— 处理设计变更全流程
```
