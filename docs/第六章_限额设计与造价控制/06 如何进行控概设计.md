### 6.1 控概设计的工作流程

```
控概设计工作流程：

  第一步：确定总投资控制目标
     │     （由甲方确定或由可研批复确定）
     ▼
  第二步：分解到各分部分项
     │     （建安 → 土建/安装/装修/专项 → 各专业分项限额）
     ▼
  第三步：各专业按限额进行设计选型
     │     （材料做法/系统方案/设备选型须在限额范围内）
     ▼
  第四步：阶段性概算核对
     │     （初设概算 → 施工图预算 → 逐级核对是否超限）
     ▼
  第五步：超限调整
           （超限部分由项目负责人组织各专业优化，直到满足限额）
```

### 6.2 各专业控概设计要点

#### 建筑专业

| 控概方向 | 具体操作 |
|---------|---------|
| 面积控制 | 严格按批复建设规模设计，不擅自超面积 |
| 做法选择 | 按品牌分档对应选择材料，不超标准 |
| 标准化 | 护理单元/诊室/病房尽量标准化设计，减少异形和特殊做法 |
| 地下室优化 | 地下室面积不超必要范围，层高不超必要值 |

#### 结构专业

| 控概方向 | 具体操作 |
|---------|---------|
| 含钢量/含混凝土量 | 设计完成后核算指标，超标准须优化 |
| 基础优化 | 多方案比选（天然地基/桩基/复合地基） |
| 梁柱优化 | 合理调整截面尺寸和配筋率 |

#### 水电暖专业

| 控概方向 | 具体操作 |
|---------|---------|
| 系统方案比选 | 重大系统方案须做≥2个方案的技术经济对比 |
| 设备选型 | 参照品牌分档库选型 |
| 管材选型 | 按区域功能选择合适管材等级 |

#### 各专项设计

| 控概方向 | 具体操作 |
|---------|---------|
| 分包概算审核 | 各专项分包须提交概算/报价，设计院审核其合理性 |
| 限额分配 | 总造价分解后给各专项一个限额，分包在限额内设计 |
| 方案比选 | 物流系统、智能化系统等须做方案比选后确定 |

### 6.3 ★概算编制中的常见遗漏项

| 遗漏项 | 后果 | 预防措施 |
|-------|------|---------|
| 专项工程费用 | 净化/防护/物流等专项未单列 | 概算中须逐项列出各专项费用 |
| 二次精装修费用 | 概算只含基础装修，精装费用未列 | 分区域单列精装概算 |
| 室外工程 | 道路/管网/景观/围墙未充分估算 | 按占总投资5-8%估算 |
| 设备安装调试费 | 只列设备采购费，未列安装调试 | 安装费一般为设备费的10-15% |
| 信息化系统 | 只列弱电硬件，未列软件和集成 | 信息化须单独列项 |
| ★大型设备特殊基础 | MRI/直线加速器的特殊结构未列 | 须在结构概算中单列 |
| 夜间施工/冬雨季措施 | — | 按地方规定列措施费 |
