### 4.1 需提前确认事项

| 事项 | 确认方 |
|------|-------|
| 市政给水接入条件（水源/水压/管径） | 自来水公司 |
| 市政排水接入条件（雨/污分流/合流、接口标高） | 市政排水 |
| 院内中水/雨水回收是否设置 | 甲方/绿建要求 |
| 太阳能热水系统方案 | 甲方/节能要求 |
| 污水处理站规模 | 甲方/环评 |
| 纯水/直饮水系统方案 | 甲方 |

### 4.4 错题本

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 管道未先落图就排卫生间 | 管道与洁具冲突 | ★铁律 |
| 核医学排水未接衰变池 | 放射性废水直排，违法 | 核医学全部排水接衰变池 |
| 屋面排水管经过电梯机房 | 漏水导致电气故障 | 管道绕行 |
| 消防前室消火栓影响电梯门 | 消防验收不通过 | 布置时核对电梯门开启范围 |
