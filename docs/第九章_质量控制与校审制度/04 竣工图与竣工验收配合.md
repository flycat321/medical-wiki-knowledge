### 4.1 竣工图编制

| 要求 | 说明 |
|------|------|
| 编制依据 | 施工图+全部设计变更+施工中的修改记录 |
| 编制时机 | 各系统安装完成后即开始编制，不等竣工后才开始 |
| ★变更整合 | 所有变更须逐条整合到竣工图中，不得遗漏 |
| 签章 | 竣工图须注明"竣工图"字样 |
| 份数 | 按合同约定提交份数 |

### 4.2 竣工验收设计院配合事项

| 验收项 | 设计院配合内容 | 负责专业 |
|-------|-------------|---------|
| 主体结构 | 确认结构施工是否符合设计 | 结构 |
| 消防验收 | 配合消防检测和验收 | 各专业 |
| 节能验收 | 提供节能设计说明和计算书 | 建筑+暖通 |
| ★辐射防护验收 | 配合第三方检测 | 建筑 |
| ★净化验收 | 配合洁净度/压差检测 | 暖通+建筑 |
| 电梯验收 | 配合特种设备检验 | 建筑+结构+电气 |
| 无障碍验收 | 确认无障碍设施合规 | 建筑 |
| 智能化验收 | 系统联调联试 | 电气 |
| 绿建验收 | 提供绿建设计和运营相关资料 | 各专业 |
| 竣工资料 | 提供全套设计文件（含竣工图） | 全专业 |

### 4.3 ★验收常见问题

| 问题 | 处理方式 |
|------|---------|
| 消防验收不通过 | 设计院排查是设计问题还是施工问题，出整改方案 |
| 节能验收不通过 | ★曲江新区教训：节能计算须经绿建部门+审图公司双审后再施工 |
| 净化检测不合格 | 暖通专业配合排查风量/压差/密封性 |
| 辐射检测超标 | 核查防护厚度是否按设计施工，出补救方案 |
| 无障碍不达标 | 核查设计图是否合规，补充整改 |
