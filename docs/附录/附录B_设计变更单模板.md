# 附录B 设计变更单模板

---

## 使用说明

- 编号规则：`项目代码-DCI-年月-序号`（如 `JN01-DCI-2603-015`）
- 变更等级：A类（>50万）/ B类（5-50万）/ C类（<5万）
- 审批权限：A类三方审批 / B类设计院+总包 / C类设计院内部
- 本模板每次变更填写一份，项目经理编号归档
