# 5.19 弱电智能化与信息化专项设计管理细则

---

## 一、概述

医疗建筑弱电智能化系统数量可达 **30-50个子系统**，远超普通公建，且与医疗业务深度耦合。

**核心特殊性：**
- 系统数量多、接口复杂（普通写字楼约15个子系统，三甲医院可达40+）
- ★与HIS/PACS/LIS等医疗信息化系统必须预留接口
- 部分系统直接关乎患者安全（护士呼叫、手术监控、婴儿防盗）
- 网络安全等级保护要求高（三甲医院通常为等保三级）

---

## 二、策划阶段

### 2.1 ★医疗建筑弱电智能化系统分类大表

#### A. 建筑智能化系统（BA/安防/管理）

| 系统 | 说明 | 医院特殊要求 |
|------|------|------------|
| 楼宇自控（BA/BMS） | 空调、照明、电梯等设备监控 | ★净化空调压差/温湿度须接入BA |
| 安防监控（CCTV） | 视频监控，覆盖出入口/走廊/电梯/停车场 | ★药房、太平间、手术室走廊须监控 |
| 入侵报警 | 周界防范、重要区域 | 药库、财务、放射源库 |
| 门禁系统 | 刷卡/人脸识别/密码 | ★手术室/ICU/药库/NICU/太平间 |
| 停车管理 | 车牌识别、缴费引导 | ★急诊通道免识别抬杆 |
| 公共广播（PA） | 背景音乐/紧急广播/分区呼叫 | 消防广播联动 |
| 信息发布 | 大厅/走廊/电梯厅LCD屏 | ★与排队叫号系统联动 |
| 会议系统 | 报告厅/会议室 | 远程会诊功能 |
| 能耗监控 | 水电气能耗分项计量 | 绿建要求 |
| 机房环境监控 | 温湿度/漏水/UPS状态 | — |

#### B. ★医疗专用系统

| 系统 | 说明 | 关键要求 |
|------|------|---------|
| ★护士呼叫系统 | 患者按钮→护士站主机→走廊显示屏 | 每个床位/卫生间/走廊均需点位 |
| ★手术示教系统 | 手术室摄像→示教室/远程直播 | 4K摄像、全景、术中导航接口 |
| ★ICU探视对讲 | 家属→患者可视对讲 | 每床位1套 |
| ★婴儿防盗系统 | RFID/红外监测婴儿位置 | 产科/NICU必配 |
| ★排队叫号系统 | 挂号→候诊→叫号→评价 | 与HIS系统对接 |
| ★分诊导引系统 | 智能分诊台+导航屏 | 与排队叫号联动 |
| ★医用时钟系统 | 全院统一授时（GPS/北斗+NTP） | 手术室/急诊/产房时间必须精确 |
| ★手术室面板系统 | 温湿度/气体/计时/音乐控制 | 集成在手术室墙面面板 |
| ★输液监控系统 | 输液余量→护士站报警 | 智慧病房选配 |
| ★床旁交互系统 | 患者床旁屏（点餐/呼叫/影音/宣教） | 智慧病房选配 |
| ★体征采集系统 | 监护仪数据→HIS/EMR | 智慧病房选配 |

#### C. 信息化系统（对弱电布线的前置要求）

| 系统 | 弱电须预留的条件 |
|------|----------------|
| HIS（医院信息系统） | 各科室信息点、网络接口 |
| PACS（影像归档传输） | ★影像科/手术室高带宽光纤接口（≥万兆） |
| LIS（检验信息系统） | 检验科信息点 |
| RIS（放射信息系统） | 放射科信息点 |
| EMR（电子病历） | 各诊室/病房信息点 |
| 自助服务终端 | 大厅/各楼层：电源+网络+排水（部分含缴费功能） |

> **★ 界面划分：** 弱电做管线（桥架、线缆、面板）和硬件（交换机、服务器机柜）；信息化公司做软件和系统集成。两者须在初设阶段明确。

#### D. 专用通信

| 系统 | 说明 |
|------|------|
| WiFi全覆盖 | 患者WiFi+医疗专网WiFi（双网隔离） |
| 5G室分 | 远程手术/远程会诊/应急通信 |
| 无线对讲 | 保安/后勤/急救 |
| 有线电话 | 各科室内线+外线 |

### 2.2 需院方确认的事项

- [ ] 智慧医院建设等级目标（国家智慧医院评级0-5级）
- [ ] 各子系统取舍（哪些做哪些不做）
- [ ] 信息化公司是否已确定（影响弱电接口需求）
- [ ] 网络安全等级保护要求（通常等保三级）
- [ ] 机房位置和面积需求
- [ ] 是否需要远程会诊/5G手术室

### 2.3 造价参考

| 系统分类 | 500床医院参考造价 | 说明 |
|---------|-----------------|------|
| 建筑智能化（A类全部） | 800-1,500万 | — |
| 医疗专用系统（B类全部） | 500-1,200万 | 视智慧化程度 |
| 信息化基础设施（C类布线+机房） | 600-1,000万 | 不含软件系统 |
| 专用通信（D类） | 200-500万 | — |
| **合计** | **2,100-4,200万** | 约占总投资3-5% |

## 三、初步设计阶段

### 3.1 建筑专业

| 配合事项 | 要求 |
|---------|------|
| 核心机房位置 | 建筑面积≥100m²（500床），宜设在建筑中部，远离水源和振动 |
| 核心机房层高 | 净高≥3.5m（含活动地板400mm+吊顶内空间） |
| 楼层弱电间 | 每层每个分区设1个，面积≥6m²，上下对齐 |
| 弱电井 | 尺寸≥1200×800mm，上下贯通，每层设检修门 |
| 桥架路由 | 走廊吊顶内预留桥架空间（强弱电分开≥300mm） |
| 室外管网 | 室外弱电管沟/管井预留 |

### 3.2 结构专业

| 配合事项 | 要求 |
|---------|------|
| 机房荷载 | 服务器机柜区≥8kN/m²，UPS电池区≥12kN/m² |
| 活动地板 | 机房设活动地板，荷载+标高需结构确认 |
| 桥架吊挂 | 重载桥架区域须预留吊挂点 |

### 3.3 暖通专业

| 配合事项 | 要求 |
|---------|------|
| 机房空调 | 精密空调，温度22±2°C，湿度45±10%RH |
| 机房新风 | 正压维持，新风量按人员计算 |
| 气体灭火后排风 | 机房灭火后须排出灭火气体（七氟丙烷等） |

### 3.4 电气专业

| 配合事项 | 要求 |
|---------|------|
| UPS配置 | 核心机房UPS≥30min后备（建议1h） |
| 双路供电 | 核心机房为一级负荷 |
| 接地 | 机房设专用接地（接地电阻≤1Ω），与防雷接地共用接地体 |
| 防雷 | SPD分级防护（总配→楼层→机房→设备端） |

## 四、施工图设计阶段

### 4.1 关键设计要点

#### ★护士呼叫系统

| 点位 | 设置要求 |
|------|---------|
| 病床 | 每床1个呼叫按钮（床头墙面/拉绳式）|
| 卫生间 | 每间1个防水呼叫按钮（距地400-600mm拉绳式）|
| 走廊 | 每组护理单元走廊设显示屏（显示呼叫床号+房号）|
| 护士站 | 呼叫主机1台，含声光报警 |
| 医生办公室 | 分机（可选） |
| ★注意 | 呼叫系统点位须与建筑平面同步——每增减一张床位，呼叫点位同步变化 |

#### ★手术示教系统

| 组件 | 要求 |
|------|------|
| 术野摄像头 | 4K全景1台+术野特写1台/手术室 |
| 编码器 | 每间手术室1套，支持4K@60fps |
| 示教室 | 大屏+音响+双向语音 |
| 远程会诊 | 支持互联网远程连接（加密） |
| ★接口 | 预留术中导航、机器人手术系统的视频接口 |

#### ★智慧病房

| 组件 | 要求 |
|------|------|
| 床旁交互屏 | 每床1台（10-15寸），含呼叫/点餐/宣教/影音 |
| 输液监控 | 每个输液架1个传感器，无线连接护士站 |
| 体征采集 | 蓝牙或WiFi连接的血压/血氧/体温设备 |
| 电子床头卡 | 每床1个电子墨水屏，与HIS同步 |

#### 综合布线

| 区域 | 信息点密度 | 说明 |
|------|----------|------|
| 诊室 | 4-6个/间 | 电脑+打印机+读卡器+预留 |
| 病房 | 2-4个/床 | 床旁屏+呼叫+预留 |
| 护士站 | 6-10个/工位 | 电脑×2+呼叫主机+打印+物流+预留 |
| 手术室 | 8-12个/间 | 示教+监护+麻醉+信息系统+预留 |
| 检验科 | 4-6个/工位 | 分析仪+电脑+LIS |
| ★影像科 | 光纤到房间 | PACS影像传输需≥万兆带宽 |

> **★ 医疗建筑信息点密度是普通办公的2-3倍，综合布线造价相应增加。**

### 4.2 常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 弱电井尺寸不够 | 线缆拥挤、散热差、维护困难 | 医院弱电井面积≥常规×1.5倍 |
| 影像科未铺光纤 | PACS影像传输慢，医生等待 | CT/MRI/DSA房间必须光纤到位 |
| 护士呼叫点位漏设卫生间 | 患者卫生间晕倒无法呼救 | 每间卫生间必须设拉绳式呼叫 |
| WiFi医疗专网与患者网未隔离 | 网络安全风险 | 物理隔离或VLAN隔离 |
| 机房面积不够 | 设备放不下、散热差 | 核心机房≥100m²（500床） |
| 弱电桥架与强电桥架距离不够 | 电磁干扰导致信号问题 | 间距≥300mm，交叉处90°垂直 |

## 五、分包方管控

### 5.1 弱电总包与子系统分包关系

```
典型模式：

  弱电总包 ──→ 综合布线、安防监控、门禁、停车、广播、BA
       │
       ├── 医疗专用分包 ──→ 护士呼叫、手术示教、婴儿防盗、时钟
       │
       ├── 信息化分包 ──→ HIS/PACS/LIS软件+集成
       │
       └── 通信运营商 ──→ WiFi/5G

  ★ 设计院角色：出弱电施工图（管线+点位），审核各分包深化图
```

### 5.2 设计院管控要点

- 各子系统接口协议须统一（TCP/IP、OPC、HL7等）
- 所有子系统须接入统一平台（或至少预留接口）
- ★信息安全要求须在招标文件中明确（等保三级）

## 六、施工阶段配合

| 节点 | 配合内容 |
|------|---------|
| 管线预埋 | 随主体施工预埋暗管暗盒 |
| 桥架安装 | 吊顶封板前完成并测通 |
| 设备安装 | 精装完成后安装面板、摄像头、屏幕等 |
| 系统调试 | ★各子系统单独调试→联调联试→与信息化系统对接测试 |
| 网络测试 | 全网测试（吞吐量、延迟、丢包率） |

## 七、验收配合

| 验收项 | 标准/方法 |
|-------|----------|
| 综合布线 | 全部链路测试（六类线/光纤），出测试报告 |
| 视频监控 | 全部点位图像质量、存储天数（≥30天）|
| 护士呼叫 | 全部点位呼叫→显示→应答功能测试 |
| 门禁 | 权限测试、消防联动（火灾释放）|
| 网络安全 | ★等保测评（三甲医院通常等保三级） |
| 系统集成 | BA/安防/信息化联动功能验证 |
