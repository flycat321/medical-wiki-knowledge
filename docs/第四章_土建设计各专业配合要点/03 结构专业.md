### 3.1 需提前沟通由业主及总包确定事项

| 事项 | 说明 |
|------|------|
| 新技术应用 | ★阻尼器/减隔震技术应用造价增加，须在可研/初设概算中考虑 |
| 基坑支护方案 | 通常由甲方或施工单位负责 |
| 装配式方案 | 预制率/装配率对结构节点的影响 |

### 3.2 主体设计要点

| 要点 | 要求 | 说明 |
|------|------|------|
| 图纸表达 | 图纸需表达内容应当明确完整 | — |
| 设备基础 | ★应提前与各专业沟通设备重量、布置位置及基础形式 | 若提前无法确定，图中明确"待设备厂家确定后配合" |
| 荷载 | 种植屋面荷载须景观和建筑专业尽早提交 | — |
| ★降板 | 防护区域降板多少？50mm够不够？视防护预评价定 | CT运输路径走道≥2.4m，转弯半径须模拟 |
| 后开洞 | 若施工图完成后需后开洞，应标注后开洞做法、是否需补强加固 | — |
| 构造柱 | 应在结构图中完善，与建筑专业核对无误后出图 | ★注意是否与门窗冲突 |
| 造型钢结构 | 较大的造型钢结构须在主体设计时完成 | 分包的由结构工程师指定配合单位并负责协调 |
| 大小雨棚 | 原则应由结构专业出图 | 无施工单位配合时由结构完成或分包，结构负责审核 |

### 3.3 项目过程配合

| 配合事项 | 要求 |
|---------|------|
| 电梯 | 确定厂家后配合出井道圈梁及构造柱图 |
| 箱式物流及垃圾物流 | 确定厂家后配合出井道圈梁及构造柱图 |
| 雨水模块 | 配合看各类井道做法砖砌还是混凝土，盖板荷载是否满足 |
| ★各专业提洞 | 暖通及各专业提洞一定要在结构图纸中出图（时间紧急也不行） |

### 3.4 错题本

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 新技术未纳入概算 | 后期超概 | 方案/可研阶段即纳入 |
| 设备基础未提前沟通 | 基础配筋不足或位置不对 | 各专业尽早提设备参数 |
| 各专业提洞未在结构图中出图 | 后开洞影响结构安全 | ★铁律：必须在结构图中出图 |
| 构造柱与门窗冲突 | 施工返工 | 出图前与建筑核对 |
