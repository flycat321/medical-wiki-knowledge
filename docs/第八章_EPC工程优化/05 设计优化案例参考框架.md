### 5.1 优化案例记录模板

每完成一项设计优化，应按以下格式记录归档：

| 栏目 | 内容 |
|------|------|
| 项目名称 | — |
| 优化编号 | YH-年月-序号 |
| 优化类型 | 结构/机电/装修/专项 |
| 原方案 | 描述+图纸编号 |
| 优化方案 | 描述+图纸编号 |
| 节约金额 | 万元 |
| 功能影响 | 是否影响使用功能（无影响/轻微/需甲方确认） |
| 审批 | 项目负责人/甲方签字 |

### 5.2 典型优化方向汇总

| 优化方向 | 潜在节约(万元/1000床) | 难度 | 说明 |
|---------|---------------------|------|------|
| 地下室面积缩减100m² | 80-120 | 中 | 需重新优化停车布局 |
| 结构含钢量降5kg/m² | 150-250 | 高 | 需结构专项优化 |
| 手术室等级降配（2间Ⅰ→Ⅲ） | 60-100 | 低 | 须甲方同意 |
| 智能化系统精简5个子系统 | 100-200 | 低 | 须甲方同意 |
| 幕墙开窗率降5% | 50-100 | 中 | 须兼顾节能和立面 |
| 走廊净宽减0.1m | 30-60 | 低 | 须满足规范底线 |
| 净化区域面积缩减5% | 40-80 | 中 | 须确保功能满足 |
