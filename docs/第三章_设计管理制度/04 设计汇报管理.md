### 4.1 汇报前准备Checklist

| 检查项 | 说明 |
|-------|------|
| ★提前确认现场条件 | 是否有投影仪？是否有电脑？屏幕尺寸？音响？ |
| ★自带设备 | 即使现场有电脑，每次也要带好自己的电脑 |
| ★提前拷贝测试 | 提前拷贝汇报文件以及音视频进去测试看是否正常 |
| 打印备份 | 准备纸质版备份（防止投影故障） |
| 内部预演 | ★专项汇报人以及年轻的项目或专业负责人给领导做重要汇报前，要跟项目负责人简单过一遍汇报文件和内容，项目负责人要整体把关 |
| 明确汇报重点 | ★汇报时候一定要分清汇报主体，汇报重点是什么，汇报时间是多久，一定要把握好重点和时间 |

> **★ 原文强调：** "杜绝到现场后才拷贝文件网上发文件而大家都在等的尴尬情景。"

### 4.2 汇报纪要与跟踪

| 要求 | 说明 |
|------|------|
| 纪要编制 | 汇报结束后当天编制纪要 |
| 重点记录 | 甲方/院方的意见和决策须逐条记录 |
| 责任分解 | 每条意见明确责任人和完成时限 |
| 跟踪落实 | 项目经理跟踪每条意见的落实情况 |
| 归档 | 纪要+签到表存入项目资料夹 |
