# 5.03 标识VI导视设计管理细则

---

## 一、概述

医疗建筑标识系统直接影响患者就医体验。在焦虑、疼痛、不熟悉环境的状态下，患者对标识的依赖远超普通公建。

**标识系统四级体系：**

| 级别 | 内容 | 位置 |
|------|------|------|
| 一级 | 院区总索引、楼栋标识、车行/人行引导 | 室外/大厅 |
| 二级 | 楼层索引、走廊方向指引、吊牌 | 各楼层公共区 |
| 三级 | 科室门牌、诊室编号、病房号 | 门旁/墙面 |
| 四级 | 温馨提示、安全标识、设备标识 | 各处 |
| ★功能性 | 分诊色彩体系、感控引导、无障碍标识、急救通道 | 全院 |

---

## 二、策划阶段

### 2.1 需院方确认的事项

- [ ] 院徽/VI规范（标准色、辅助色、字体、院徽使用规范）
- [ ] 中英文对照要求（是否加拼音、是否需多语种）
- [ ] ★分诊色彩体系（是否按科室分色，如内科蓝、外科绿、妇产粉等）
- [ ] 数字标识vs实体标识的比例（电子屏占比）
- [ ] 标识材质偏好（金属/亚克力/搪瓷/发光）
- [ ] 室外标识是否含精神堡垒

### 2.2 造价参考

| 规模 | 参考造价 | 说明 |
|------|---------|------|
| 500床医院 | 200-500万 | 含室内外全部标识 |
| 1000床医院 | 400-800万 | — |

## 三、设计阶段各专业配合

### 3.1 建筑专业

| 配合事项 | 要求 |
|---------|------|
| 标识安装位置 | 方案阶段即在立面/平面中预留标识位置 |
| 墙面基层 | 标识安装处墙面须平整、有埋件或安装条件 |
| 吊顶 | 吊装式标识须确认吊顶承重和固定方式 |

### 3.2 电气专业

| 配合事项 | 要求 |
|---------|------|
| 发光标识供电 | 每处发光标识预留220V电源 |
| LED屏 | 各楼层信息发布屏供电+网络 |
| 夜间照明 | 室外标识泛光照明配合 |

### 3.3 精装修配合

- 标识风格须与精装风格统一
- 标识安装面的材质须配合（石材面/乳胶漆面/玻璃面安装方式不同）

### 3.4 ★与弱电系统联动

| 联动系统 | 方式 |
|---------|------|
| 排队叫号 | 叫号屏与标识导视统一设计风格 |
| 信息发布 | 各楼层LCD屏内容含导视信息 |
| 分诊导引 | 智能导诊屏与实体标识互补 |

## 四、分包方管控

| 阶段 | 分包须提交 | 设计院审核要点 |
|------|----------|--------------|
| 方案 | 标识系统策划方案、色彩方案、材质样板 | 与建筑风格一致性、规范合规性 |
| 深化 | 全部标识定位图、加工图 | 安装位置与建筑/精装图一致 |
| 施工 | 安装方案 | 不破坏防水/防火/结构 |

---

## 五、★医疗建筑标识特殊要求

| 要求 | 说明 |
|------|------|
| 无障碍标识 | 盲文触摸标识、轮椅坡道标识、无障碍卫生间标识 |
| 感控引导 | 洗手提示、口罩佩戴提示、隔离区域警示 |
| 急救通道 | 120急救车通道标识须醒目、与消防通道区分 |
| ★分诊色彩 | 建议各科室/楼层采用不同辅助色，帮助患者区分 |
| 夜间可识别 | 发光标识/反光标识确保夜间可读 |
| 防碰撞 | 吊装标识底部高度≥2200mm |

---

## 六、常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 标识定位图与精装竣工图不符 | 标识安装在错误位置 | 标识安装前须与竣工图校核 |
| 标识遮挡消防设施 | 消防验收问题 | 定位时避开消火栓箱/灭火器箱/疏散指示 |
| 标识字体过小 | 老年患者无法识读 | 一级标识字高≥50mm、二级≥30mm、三级≥20mm |
| 标识信息与HIS科室名称不一致 | 患者困惑 | ★标识科室名称须与HIS系统完全一致 |
| 发光标识未预留电源 | 后期明装线缆 | 电气专业在施工图阶段即预留 |
