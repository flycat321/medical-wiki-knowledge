# 5.08 景观设计管理细则

---

## 一、概述

医疗建筑景观不仅是美化，更承担**疗愈功能**。循证设计研究表明，接触自然景观可缩短住院时间、减轻疼痛感知、降低焦虑水平。

---

## 二、施工图前项目负责人须明确的6项事项

> **以文件形式确认：**

| 序号 | 事项 | 说明 |
|------|------|------|
| 1 | 设计定位 | 景观风格、档次 |
| 2 | 造价 | 景观工程造价控制目标 |
| 3 | 所有地面屋面材质、做法、树种 | 须附相关资料清单说明 |
| 4 | 与室外总体专业的界面划分 | 道路/管网/竖向由谁做 |
| 5 | 标识专业室外用电点位 | 景观需配合预留 |
| 6 | ★种植屋面 | 方案阶段要考虑层高，施工图阶段需结构配合荷载 |

---

## 三、★医疗建筑景观特殊要求

### 3.1 康复花园/疗愈景观

| 要点 | 要求 |
|------|------|
| 无障碍路径 | 全程无台阶，坡道≤1:12，路面防滑 |
| 轮椅友好 | 路面宽度≥1.8m，转弯处≥2.0m |
| 感官花园 | 视觉（花卉色彩）+嗅觉（芳香植物）+触觉（质感差异）+听觉（水景/风铃）|
| 安全 | 不使用有毒植物、不设跌落风险水景 |
| 遮荫 | 充足的乔木遮荫和廊架（患者户外活动需求）|
| ★ICU/病房可视 | 病房/ICU窗户朝向景观（循证设计核心原则）|

### 3.2 120急救通道

| 要点 | 要求 |
|------|------|
| 通道宽度 | ≥4.0m |
| 转弯半径 | ≥12m |
| 承重 | 消防车/救护车荷载 |
| 标识 | 醒目标识、夜间照明 |
| ★不得被景观遮挡 | 乔木种植不得影响急救车通行净高≥4.0m |

## 四、内部配合

| 配合事项 | 要求 |
|---------|------|
| 与建筑屋面做法 | 景观屋面做法应与建筑屋面做法保持一致 |
| 与建筑地面做法 | 室外广场铺装与建筑首层地面标高/做法衔接 |
| ★种植屋面荷载 | 方案阶段即告知结构专业种植层厚度和荷载（湿土≥8-15kN/m²）|
| 给排水 | 景观浇灌用水接口、排水暗沟 |
| 电气 | 景观照明、草坪灯、标识灯供电 |

---

## 五、常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 种植屋面荷载未提结构 | 结构楼板承载力不够，后期加固 | ★方案阶段确定种植范围，同步提荷载 |
| 急救通道被景观遮挡 | 救护车无法通行 | 急救通道两侧种植净高≥4.0m |
| 景观排水与建筑排水冲突 | 积水 | 竖向设计须与室外总体统一 |
| 康复花园使用有毒植物 | 安全风险（精神科/老年科患者）| 植物选型须安全审查 |
