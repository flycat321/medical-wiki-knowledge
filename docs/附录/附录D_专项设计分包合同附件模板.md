# 附录D 专项设计分包合同附件模板

---

## 使用说明

每个专项设计分包合同签订时，须将本模板中对应专项的附件内容纳入合同。项目负责人须在分包启动前逐项确认。

| 招标图 | 招标深度图纸+工程量清单 | 按需 | 设计院+总包 |

### 附件4：审核与修改流程

```
分包设计成果审核流程：

  分包提交设计成果 → 设计院对口负责人初审（5个工作日）
       → 设计院各相关专业会审（3个工作日）
       → 审核意见反馈分包 → 分包修改 → 再次审核
       → 审核通过 → 设计院盖审核章 → 可用于施工

  ★ 未经设计院审核盖章的分包图纸不得用于施工
```

## 二、各专项分包特殊附件

### 2.1 精装修分包

**须在施工图前项目负责人明确的6项事项（作为合同附件）：**

| 序号 | 事项 | 确认内容 | 确认人 | 确认日期 |
|------|------|---------|-------|---------|
| 1 | 设计定位 | 风格/色彩/档次 | | |
| 2 | 造价控制要求 | 分区域限额 | | |
| 3 | 墙顶地材料规格及收边节点 | 附资料清单 | | |
| 4 | ★与净化/防护/实验室界面划分 | 哪些区域精装做/净化做 | | |
| 5 | 固定家具与活动家具界限 | 列清单 | | |
| 6 | 消防专项是否包含 | 是/否 | | |

### 2.2 净化分包

**合同附件须包含：**

| 附件 | 内容 |
|------|------|
| 净化区域策划表 | 各房间洁净等级/压差/换气次数/温湿度 |
| 界面划分清单 | ★土建/一次机电/净化分包各自范围 |
| 冷热源接口条件 | 设计院暖通提供的冷冻水/热水参数及接口位置 |
| 净化装修材料标准 | 品牌/等级/颜色要求 |
| 验收标准 | 洁净度/压差/温湿度/噪声检测标准 |

### 2.3 防辐射分包

**合同附件须包含：**

| 附件 | 内容 |
|------|------|
| 放射设备清单 | 品牌型号/参数/安装楼层/机房编号 |
| 防护预评价报告 | 各机房各面防护当量 |
| 界面划分清单 | ★防护公司做防护，设计院做土建和精装 |
| 预埋件协调方案 | 铅门门框预埋时间节点 |

### 2.4 幕墙分包

**合同附件须包含：**

| 附件 | 内容 |
|------|------|
| 建筑立面设计图 | 作为幕墙深化的基准 |
| 暖通外墙留洞清单 | ★所有外墙外窗留洞 |
| 防雷接地要求 | 电气提供 |
| 节能性能要求 | K值/遮阳系数等 |

### 2.5 景观分包

**施工图前项目负责人须明确的事项（作为合同附件）：**

| 序号 | 事项 | 确认内容 | 确认人 | 确认日期 |
|------|------|---------|-------|---------|
| 1 | 设计定位 | 风格/档次 | | |
| 2 | 造价 | 景观工程限额 | | |
| 3 | 地面屋面材质做法树种 | 附资料清单 | | |
| 4 | 与室外总体的界面划分 | 道路/管网/竖向由谁做 | | |
| 5 | 标识室外用电点位 | 景观配合预留 | | |
| 6 | ★种植屋面 | 方案阶段层高/施工图阶段结构荷载 | | |

### 2.6 物流系统分包

**合同附件须包含：**

| 附件 | 内容 |
|------|------|
| 物流系统策划表 | 系统类型/站点数量/站点位置 |
| 结构留洞图 | 垂直通道留洞尺寸和位置 |
| ★通用性原则声明 | 结构留洞须满足后续不同厂家招标需求 |
| 配电需求 | 各站点/主机电力需求 |

### 2.7 弱电智能化分包

**合同附件须包含：**

| 附件 | 内容 |
|------|------|
| 智能化系统清单 | 各子系统取舍确认 |
| ★与信息化公司界面划分 | 弱电做管线硬件/信息化做软件集成 |
| 机房条件 | 面积/荷载/空调/电力 |
| 等保要求 | 网络安全等级保护等级 |

### 2.8 厨房分包

**合同附件须包含：**

| 附件 | 内容 |
|------|------|
| 建筑划定的厨房区域平面 | 分包在此基础上深化 |
| ★运营公司参与声明 | 提示甲方引入运营公司 |
| 设备清单 | 分包提供 |
| 界面划分 | 明沟在建筑图中表达/分包提供位置 |
