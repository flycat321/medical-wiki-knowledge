# 5.17 太平间专项设计管理细则

---

## 一、概述

太平间设计的核心原则：**流线独立、视线遮蔽、情感关怀**。这是医院中最敏感的空间之一，设计质量直接影响医院运营和家属体验。

---

## 二、功能组成

| 功能房间 | 面积参考 | 说明 |
|---------|---------|------|
| 遗体接收/暂存 | 15-25m² | 与病区电梯直通 |
| 冷藏间 | 20-40m² | 冷藏柜数量按床位数估算（500床约8-12位）|
| 告别/瞻仰厅 | 30-50m² | 家属告别空间 |
| 化妆整容室 | 15-20m² | 遗体整理 |
| 家属等候区 | 20-30m² | ★情感化设计，安静、柔和 |
| 办公室 | 10-15m² | 管理人员 |
| 储藏/设备间 | 10-15m² | 棺材/寿衣存放 |

---

## 三、设计要点

### 3.1 建筑专业

| 要点 | 要求 |
|------|------|
| ★流线独立 | 太平间须有独立出入口，不与门诊/住院区域共用通道 |
| ★视线遮蔽 | 遗体运送路线不得经过患者/家属活动区域 |
| ★专用电梯 | 从住院楼至太平间须有专用电梯或在非诊疗时段使用专用电梯 |
| 独立出入口 | 殡仪车辆停靠处须遮蔽，不面对主入口和病房窗户 |
| 无障碍 | 家属告别区须无障碍（轮椅通行、盲道）|
| 标识 | 低调标识，不使用醒目色彩 |

### 3.2 暖通专业

| 要点 | 要求 |
|------|------|
| ★负压通风 | 冷藏间、化妆室须负压通风 |
| 除臭 | 排风经活性炭过滤+紫外消毒后高空排放 |
| 温度 | 冷藏间环境温度≤18°C |
| 告别厅 | 正常空调，温度适宜 |

### 3.3 给排水专业

| 要点 | 要求 |
|------|------|
| 排水 | ★特殊排水（含消毒处理后排放）|
| 给水 | 遗体清洗用水，冷热水均需 |
| 消毒 | 排水须经消毒处理 |

### 3.4 电气专业

| 要点 | 要求 |
|------|------|
| 照明 | 冷藏间≥100lx、告别厅可调光（柔和氛围）|
| 监控 | ★冷藏间须24h监控 |
| 应急电源 | 冷藏设备须接应急电源 |

### 3.5 冷藏设备

| 参数 | 要求 |
|------|------|
| 冷藏温度 | -5°C~4°C |
| 数量估算 | 500床医院约8-12位 |
| 设备类型 | 抽屉式冷藏柜（最常见）|

---

## 四、★情感化设计要点

| 要点 | 说明 |
|------|------|
| 家属等候区 | 安静、私密，设座椅/饮水/纸巾 |
| 告别厅 | 灯光柔和可调，设花台/背景墙/音响（轻音乐）|
| 通道 | 从等候区到告别厅的路径须平静、无嘈杂 |
| 出口 | 家属离开的出口不得再经过冷藏/遗体区域 |
| ★殡仪车停靠 | 遮蔽设计（雨棚+围墙），不面对病房和公共区域 |

---

## 五、常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 太平间与住院区共用电梯 | 患者/家属心理不适 | ★设专用电梯或专用时段 |
| 殡仪车停靠位面对病房 | 住院患者看到，影响心理 | 遮蔽设计 |
| 通风不足 | 异味扩散 | 负压+除臭+高空排放 |
| 冷藏设备未接应急电源 | 停电导致设备停运 | 接柴发或UPS |
