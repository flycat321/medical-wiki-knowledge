以下是医疗建筑EPC项目前期**必须在特定时间节点前完成的关键决策**，逾期将导致后续设计返工或重大变更：

| 决策事项 | 最迟完成时间 | 影响范围 | 逾期后果 |
|---------|------------|---------|---------|
| 建设规模（床位数/门诊量） | 可研批复前 | 全局 | 重新可研 |
| 科室设置方案 | 方案设计开始前 | 全局 | 方案推翻重来 |
| 手术室层层高 | ★方案设计期间 | 结构全局 | 手术室净化无法实施 |
| 地下室层数 | 方案设计期间 | 结构+基坑 | 基坑方案重做 |
| MRI设备型号（1.5T/3.0T） | ★方案设计期间 | 结构+屏蔽+暖通 | 结构加固/无法实施 |
| 直线加速器型号 | ★方案设计期间 | 放疗科整体结构 | 无法补救 |
| CT设备型号 | 初设阶段 | 机房条件 | 施工图变更 |
| 电梯配置方案 | 方案设计期间 | 核心筒结构 | 改井道代价大 |
| 物流系统类型 | 方案设计期间 | 井道预留 | 无法加装 |
| 装配式方案 | 初设阶段 | 结构节点 | 施工图返工 |
| 净化区域方案 | 初设阶段 | 层高+暖通 | 施工图变更 |
| 精装修定位 | 施工图开始前 | 精装全局 | 精装方案推翻 |
| 智能化系统清单 | 初设阶段 | 机房+布线 | 弱电施工图变更 |
| 各专项分包确定 | 施工图开始前 | 各专项施工图 | 施工图等待 |
