# 5.14 直升机停机坪专项设计管理细则

---

## 一、概述

三甲医院/急救中心/偏远地区医院可能需要配置直升机停机坪，用于急救转运。

---

## 二、策划阶段

### 2.1 选址方式

| 方式 | 适用场景 | 优缺点 |
|------|---------|-------|
| 屋面停机坪 | 建筑高度允许、净空条件满足 | 节省地面空间，但结构荷载大 |
| 地面停机坪 | 场地充裕、净空条件好 | 荷载问题小，但占地面积大 |

### 2.2 需确认的事项

- [ ] 起降机型（决定停机坪尺寸和荷载）
- [ ] ★净空条件（须经民航/空管部门审批，是否在航路限制区内）
- [ ] 与急诊/手术室的流线连接方案
- [ ] 夜间起降需求（影响灯光配置）

---

## 三、设计要点

### 3.1 建筑

| 参数 | 要求 |
|------|------|
| 起降区（FATO） | ≥直升机全尺寸×1.5倍（通常≥20m×20m） |
| 安全区 | FATO外侧≥3m宽安全区 |
| 地面标识 | "H"标识、边界灯、风向标 |
| ★急救通道 | 停机坪→电梯→急诊/手术室，全程无障碍，路径≤100m为佳 |
| 电梯 | 担架梯直达手术层/急诊层 |

### 3.2 结构

| 参数 | 要求 |
|------|------|
| 荷载 | 起降区≥最大起飞重量×1.5倍动荷载系数 |
| 常见机型荷载 | EC135约3吨、AW139约7吨 |
| 局部荷载 | 起落架集中荷载，须按设备参数验算 |
| 抗风 | 屋面停机坪须考虑旋翼下洗气流对周边结构的影响 |

### 3.3 消防

| 参数 | 要求 |
|------|------|
| 泡沫灭火 | 停机坪配备泡沫灭火系统（手动/自动） |
| 灭火器 | ABC干粉灭火器+泡沫灭火器 |
| 排水 | 停机坪设排水沟+油水分离器 |

### 3.4 电气

| 参数 | 要求 |
|------|------|
| 航空灯光 | 边灯、进近灯、目视进近下滑指示灯 |
| 助航设备 | 风向标照明 |
| 供电 | 航空灯光为一级负荷，须双路供电+UPS |
| 障碍灯 | 建筑周边高点设航空障碍灯 |

---

## 四、审批流程

| 步骤 | 主管部门 | 说明 |
|------|---------|------|
| 1. 净空审查 | 民航地区管理局 | ★是否在机场净空限制区内 |
| 2. 空域审批 | 空军/民航 | 起降空域使用许可 |
| 3. 停机坪设计审查 | 民航局/住建 | 按CCAR-155或MH5013 |
| 4. 竣工验收 | 同上 | — |

> **★ 审批周期长（6-12个月），须在项目立项时即启动。**
