### 2.1 变更分级

| 级别 | 定义 | 造价影响 | 审批权限 | 处理时限 |
|------|------|---------|---------|---------|
| A类（重大） | 涉及功能布局调整/结构体系变化/增减面积 | >50万元 | 甲方+总包+设计院三方审批 | 7个工作日 |
| B类（一般） | 涉及材料/做法/局部尺寸调整 | 5-50万元 | 设计院+总包审批 | 3个工作日 |
| C类（轻微） | 尺寸微调/文字修正/补充说明 | <5万元 | 设计院内部审批 | 1个工作日 |

### 2.2 变更界定

| 变更来源 | 界定 | 费用承担 |
|---------|------|---------|
| 甲方新增需求 | 甲方提出原合同范围外的要求 | 甲方承担（设计费+施工费） |
| 设计优化 | 设计院主动提出的优化方案（降造价/提质量） | 不增加设计费 |
| 设计错误/遗漏 | 设计院自身原因导致的修改 | 设计院承担设计修改费 |
| 施工偏差 | 施工与图纸不符导致的设计配合 | 施工单位承担 |
| 规范变更 | 新规范实施导致的设计调整 | 协商确定 |
| 不可预见 | 地质条件变化等客观原因 | 协商确定 |

### 2.3 变更格式与编号

**编号规则：** `项目代码-DCI-年月-序号`

- DCI = Design Change Instruction（设计变更指令）
- 示例：`JN01-DCI-2603-015`（济南01项目2026年3月第15号变更）

**变更单内容须包含：**

| 栏目 | 内容 |
|------|------|
| 变更编号 | 按编号规则 |
| 变更等级 | A/B/C |
| 变更原因 | 甲方需求/设计优化/设计修正/规范变更/其他 |
| 变更内容 | 文字描述+标注图（必要时附修改前后对比图） |
| 涉及专业 | 列出所有受影响的专业 |
| 造价影响 | 增/减金额估算（A/B类必填） |
| 工期影响 | 是否影响施工进度（A类必填） |
| 审批签字 | 按分级审批权限签字 |

### 2.4 变更流程

```
变更流程（EPC模式）：

  变更发起（甲方/设计院/施工单位/分包）
       │
       ▼
  项目经理登记、编号、判定等级
       │
       ▼
  项目负责人初审（判断合理性、影响范围）
       │
       ├── C类：项目负责人批准 → 执行 → 归档
       │
       ├── B类：设计院+总包审批 → 执行 → 归档
       │
       └── A类：三方会审 → 甲方批准 → 执行 → 归档

  ★ 关键原则：
  1. 口头变更无效——必须走书面流程
  2. 未经审批不得出图——先批后改
  3. 变更图须标注变更号——便于施工单位识别
  4. 项目经理须建立《变更台账》跟踪所有变更的状态
```
