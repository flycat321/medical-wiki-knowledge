### 1.1 总体架构

```
医疗建筑EPC设计项目组织架构：

  ┌──────────────────────────────────────────────────┐
  │                  项目负责人                        │
  │    （总协调、对外接口、品质把控、重大决策）          │
  └───────────────┬──────────────────────────────────┘
                  │
        ┌─────────┼─────────┐
        │         │         │
  ┌─────┴──┐ ┌───┴────┐ ┌──┴────────────┐
  │项目经理 │ │各专业   │ │专项设计负责人  │
  │(进度/成本│ │负责人   │ │（各分包对接）  │
  │/资料管理)│ │        │ │               │
  └────────┘ └───┬────┘ └──────────────┘
                  │
        ┌────┬────┼────┬─────┐
        建筑  结构  给排水 暖通  电气
        专业  专业  专业   专业  专业

  ★ 新增岗位：医疗工艺顾问/对接人
    （负责与院方科室主任沟通，可由建筑主创兼任）
```

### 1.2 项目负责人

#### 岗位职责

| 职责 | 具体内容 | 关键动作 |
|------|---------|---------|
| 总协调 | 统筹内部各专业和外部各方的工作节奏 | 主持周例会，协调资源 |
| 对外接口 | 与甲方/总包方的第一联系人 | 参加甲方会议，传达设计意图 |
| 品质把控 | 各阶段成果的最终审核人 | ★交付给甲方的图册项目负责人一定要提前看 |
| 重大决策 | 涉及方案调整/重大变更/费用增减的决策 | 书面记录决策依据 |
| 进度管控 | 总体进度控制，关键节点督办 | 每周跟踪里程碑 |
| 前期策划 | 主导医疗工艺策划、设备专题会、专项分包选择 | 详见第二章 |
| 限额设计 | 牵头制定限额设计目标，监督各专业执行 | 详见第六章 |

#### ★项目负责人的管理红线

| 红线 | 说明 |
|------|------|
| 交付物必须提前审核 | 交付给甲方的任何文件（图册/汇报/变更）必须项目负责人提前审看 |
| 重大决策必须书面记录 | 口头约定不算数，须会议纪要或邮件确认 |
| 不得跳过项目经理直接安排设计人员 | 项目经理负责任务分发和进度跟踪 |
| 专项汇报须提前过审 | ★专项汇报人以及年轻的项目或专业负责人给领导做重要汇报前，要跟项目负责人简单过一遍汇报文件和内容 |

### 1.3 项目经理

#### 岗位职责

| 职责 | 具体内容 |
|------|---------|
| 进度管理 | 编制进度计划、跟踪执行、预警延期 |
| 成本管理 | 设计工时统计、外协费用管控 |
| 资料管理 | ★建立项目资料管理体系（实体文件夹+电子资料库，实时在所内网盘更新） |
| 会议管理 | 组织内部例会、编制/分发会议纪要 |
| 提资管理 | 跟踪各专业提资/回资的完成情况 |
| 变更管理 | 变更登记、编号、流转、归档 |
| 图纸管理 | 出图登记、版本管控、交付签收 |

#### ★项目资料管理体系

```
项目资料管理体系（实体+电子双轨）：

  实体资料夹（项目办公室存放）：
  ├── 01_合同及附件
  ├── 02_前期资料（规划条件/地勘/测绘等）
  ├── 03_甲方确认文件（需求书/会议纪要/书面确认）
  ├── 04_设计成果（方案/初设/施工图各阶段存档）
  ├── 05_变更记录
  ├── 06_会议纪要
  ├── 07_提资记录
  ├── 08_专项分包管理
  ├── 09_施工配合记录
  └── 10_竣工验收

  电子资料库（所内网盘，镜像更新）：
  └── 同上目录结构
```

### 1.4 各专业负责人

#### 岗位职责

| 职责 | 具体内容 |
|------|---------|
| 专业设计质量 | 本专业设计成果的质量负责人 |
| 专业间提资 | 主动向相关专业提资，及时响应回资需求 |
| 专业内协调 | 本专业内部人员分工和进度协调 |
| 规范执行 | 确保本专业设计符合现行规范 |
| 审核 | 本专业图纸的初审（校审流程详见第九章） |
| ★错题本维护 | 项目完成后补充本专业的错题本 |

### 1.5 专项设计负责人

#### 岗位职责

| 职责 | 具体内容 |
|------|---------|
| 分包对接 | 作为设计院与专项分包单位之间的技术接口人 |
| 技术审核 | 审核分包单位提交的设计成果 |
| 界面把控 | 确保分包与土建各专业之间界面清晰 |
| 进度跟踪 | 跟踪分包设计进度，预警延期 |
| 质量控制 | 发现分包设计问题及时提出并督促整改 |

#### ★原则

> 专项设计的质量最终由设计院承担。分包出了问题，设计院不能以"这是分包做的"为由推卸责任。因此，专项设计负责人必须具备审核分包图纸的专业能力。

### 1.6 ★医疗工艺顾问/对接人（新增岗位）

| 职责 | 具体内容 |
|------|---------|
| 科室沟通 | 负责与院方各科室主任的日常沟通 |
| 需求翻译 | 将医疗需求翻译为建筑语言 |
| 流线审核 | 审核各科室内部流线和科室间流线的合理性 |
| 设备条件 | 汇总大型设备机房条件，分发至各专业 |
| 感控审查 | 审查洁污分流、医患分流的落实情况 |

> **★ 建议：** 可由建筑专业主创兼任，但必须明确为一项独立职责，不能因为出图忙而忽略。有条件的项目可聘请专业医疗工艺顾问。
