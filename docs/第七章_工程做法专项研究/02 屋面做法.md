### 2.1 正置式屋面

| 项目 | 内容 |
|------|------|
| 构造层次（自下而上） | 结构板→找坡层→找平层→防水层→保温层→保护层 |
| 优点 | 造价低、施工方便、技术成熟 |
| 缺点 | 防水层直接暴露在温度应力下，易老化；检修时需破坏保温层 |
| 适用 | 普通不上人屋面 |
| 造价参考 | 180-250元/m² |
| 注意事项 | 保温层须做好防潮处理；屋面排水坡度≥2%；泛水高度≥250mm |

### 2.2 倒置式屋面

| 项目 | 内容 |
|------|------|
| 构造层次（自下而上） | 结构板→找坡层→找平层→防水层→保温层→滤水层→保护层 |
| 优点 | 防水层受保温层保护，寿命长；适合上人屋面 |
| 缺点 | 保温层受水浸泡，须用XPS等吸水率低的材料；保温层可能位移 |
| 适用 | 上人屋面、设备平台 |
| 造价参考 | 250-350元/m² |
| 注意事项 | 保温板须压砖或用卵石压覆；排水口处保温层须断开 |

### 2.3 种植屋面

| 项目 | 内容 |
|------|------|
| 构造层次 | 结构板→找坡→找平→防水层→耐根穿刺层→排（蓄）水层→过滤层→种植土→植物 |
| 种植层厚度 | 草坪150-300mm；灌木300-600mm；小乔木≥800mm |
| ★结构荷载 | 湿土+植物+水≈8-15kN/m²，须在方案阶段提交结构 |
| 造价参考 | 400-800元/m²（不含植物） |
| 注意事项 | ★方案阶段就要确定种植范围并提结构荷载；耐根穿刺防水必须；排水板规格影响排水效果；须设检修通道（便于防水检修） |
