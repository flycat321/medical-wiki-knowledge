### 7.1 ★施工图交底

施工图交底是设计院向施工单位正式移交设计意图的关键环节，须按专业逐项进行：

| 专业/专项 | 交底重点 |
|----------|---------|
| 建筑专业 | 功能布局意图、层高控制、特殊做法、预留预埋 |
| 结构专业 | 基础形式、结构体系、关键节点、预埋件要求 |
| 给排水专业 | 系统原则、特殊管道、预留条件 |
| 暖通专业 | 系统划分、设备参数、风管路由 |
| 电气专业 | 供电方案、配电系统、消防联动 |
| 精装修专项 | 材料做法、节点要求、与土建界面 |
| 幕墙专项 | 幕墙系统、预埋件、与各专业接口 |
| 净化专项 | 洁净等级、压差、施工要求 |
| 物流专项 | 站点位置、留洞、穿楼板封堵 |
| 标识专项 | 安装位置、电气预留 |
| 防护专项 | 防护方案、铅板做法、预埋件 |
| 景观专项 | 地面做法、种植屋面荷载 |
| 总图专项 | 竖向标高、管网路由 |
| 机械车位 | 基坑、荷载、通用性 |

### 7.2 施工阶段配合原则

| 原则 | 说明 |
|------|------|
| 及时响应 | 施工单位提出的技术问题须在约定时限内回复（通常A类3天/B类2天/C类1天） |
| 书面为准 | 现场口头商定的事项须补书面确认 |
| 变更先行 | 现场需要设计调整的，先走变更流程再施工 |
| 旁站参与 | 关键施工节点（基础验槽、预埋件、隐蔽工程）设计院应到场 |
| ★不得擅自降低标准 | 施工单位提出的"替代方案"须经设计院技术评估后方可同意 |
