# 5.18 洗衣房专项设计管理细则

---

## 一、概述

医院洗衣房承担全院被服、手术衣、工作服等织物的清洗消毒，★核心是感控分区——洁污严格分离、单向流。

---

## 二、设计要点

### 2.1 ★感控分区原则

```
洗衣房"单向流"原则：

  污染织物收集 → 污染区（分拣/装机）→ 清洁区（烘干/熨烫/折叠）→ 无菌/洁净区（存放/发放）
       │              (负压)                (正压)                    (最高正压)
       │
       └── 污物通道进入（不与洁净织物出口共用）

  ★ 隔离式洗衣机（barrier washer）：前端装污物（污染侧），后端出洁净衣物（清洁侧），
    机器本身构成两区的物理隔断。
```

### 2.2 功能组成与面积

| 功能区 | 面积参考（500床） | 说明 |
|-------|-----------------|------|
| 污物接收/分拣区 | 40-60m² | 负压 |
| 洗涤区 | 60-100m² | 大型洗衣龙/隔离式洗衣机 |
| 烘干区 | 30-50m² | 高温高湿 |
| 熨烫/折叠区 | 40-60m² | — |
| 洁净存放/发放区 | 30-50m² | 正压 |
| 设备间/锅炉间 | 20-40m² | 蒸汽锅炉 |

### 2.3 各专业配合

#### 结构专业

| 配合事项 | 要求 |
|---------|------|
| 设备荷载 | ★大型洗衣龙重量可达5-10吨，须独立基础 |
| 振动隔离 | 洗脱机高速脱水时振动大，须隔振基础 |
| 降板 | 排水需要，降板80-100mm |

#### 给排水专业

| 配合事项 | 要求 |
|---------|------|
| 热水供应 | ★用水量极大（日用水量可达50-100吨/500床）|
| 蒸汽 | 烘干/熨烫需蒸汽，须设蒸汽管道 |
| 排水 | 排水量大，主排水管≥DN150，设隔油/纤维过滤 |
| 废水处理 | ★洗涤废水含碱性物质，须中和处理后排放 |

#### 暖通专业

| 配合事项 | 要求 |
|---------|------|
| ★高温高湿 | 烘干区温度高、湿度大，须强力排风 |
| 排汽 | 烘干机排汽管道直通室外 |
| 通风 | 污染区负压，清洁区正压 |
| 补风 | 排风量大，须充足补风 |

#### 电气专业

| 配合事项 | 要求 |
|---------|------|
| 大功率设备 | 洗衣龙/烘干机/熨烫机单台功率可达50-100kW |
| 供电 | 三相380V，总配电容量大 |
| 防水 | 所有电气设备须防水（IP65+）|

## 四、常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 洁污流线可逆流 | 交叉感染风险 | ★必须单向流+隔离式洗衣机 |
| 排水管径不够 | 排水不畅、地面积水 | 主排水管≥DN150 |
| 振动影响楼上 | 洗脱机振动传至上层房间 | 隔振基础+弹簧减振器 |
| 蒸汽管道未保温 | 能耗高、烫伤风险 | 蒸汽管道全程保温 |
| 排汽未引至室外 | 室内高温高湿、墙面结露 | 排汽管直通屋面 |
