### 4.1 交底原则

| 原则 | 说明 |
|------|------|
| 全覆盖 | 每个专业/专项必须交底，不得遗漏 |
| 书面化 | 交底内容形成书面记录，参与方签字 |
| 重点突出 | 标注"必须关注"的关键节点和易错点 |
| 互动式 | 施工单位提问→设计院当场或限期解答 |

### 4.2 各专业/专项交底要点清单

#### 建筑专业

| 交底内容 | 重点说明 |
|---------|---------|
| 功能布局意图 | 各科室流线关系、重点房间功能要求 |
| 层高控制 | 各层净高要求，★手术层/地下室/设备层 |
| 特殊做法 | 降板/防水/保温/隔声的特殊要求 |
| 预留预埋 | 各类预留洞/套管/埋件的汇总 |
| ★上下层关系 | 手术室/配电室上方禁止卫生间 |

#### 结构专业

| 交底内容 | 重点说明 |
|---------|---------|
| 结构体系 | 抗震等级、结构形式、关键节点 |
| 基础形式 | 桩基/天然地基/复合地基施工要点 |
| 预埋件 | ★电梯/幕墙/设备吊挂预埋件的精确定位要求 |
| 后浇带 | 位置、宽度、封闭时间 |
| ★大体积混凝土 | 直线加速器等超厚墙体的浇筑方案 |

#### 给排水专业

| 交底内容 | 重点说明 |
|---------|---------|
| 管材选型 | 不同区域管材等级 |
| ★衰变池 | 核医学排水特殊要求 |
| 消防系统 | 消火栓/喷淋/气体灭火分区 |
| 雨水系统 | 虹吸/重力排水方案 |

#### 暖通专业

| 交底内容 | 重点说明 |
|---------|---------|
| 空调系统 | 冷热源方案、水系统/风系统划分 |
| ★净化空调 | 与净化分包的界面、冷热源接口 |
| 防排烟 | 排烟分区、排烟窗/排烟风机 |
| 医用气体 | 气站位置、管道材质、终端位置 |

#### 电气专业

| 交底内容 | 重点说明 |
|---------|---------|
| 供配电 | 双路供电+柴发方案 |
| ★IT系统 | 手术室/ICU隔离供电系统 |
| 消防 | 火灾自动报警、消防联动 |
| 防雷接地 | 接地方式、接地电阻要求 |

#### 各专项

| 专项 | 交底重点 |
|------|---------|
| 精装修 | 材料做法、界面划分、样板间要求 |
| 幕墙 | 预埋件、分格方案、与各专业接口 |
| 净化 | 洁净等级、压差、施工封闭管理 |
| 物流 | 站点位置、留洞、穿楼板封堵 |
| 标识 | 安装位置、电气预留 |
| 防护 | 铅板/硫酸钡施工工艺、隐蔽验收 |
| 景观 | 地面做法、种植屋面荷载 |
| 总图 | 竖向标高、管网路由、道路做法 |
| 机械车位 | 基坑/荷载/通用性预留 |
