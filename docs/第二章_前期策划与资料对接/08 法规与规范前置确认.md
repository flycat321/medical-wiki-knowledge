### 8.1 须在设计开始前确认的法规规范

| 类别 | 内容 | 责任人 | 说明 |
|------|------|-------|------|
| 国家标准 | GB 51039《综合医院建筑设计规范》最新版 | 建筑专业 | 医院设计总纲 |
| 地方标准 | 当地医疗建筑相关地方标准/导则 | 建筑专业 | ★各地可能有特殊要求 |
| 消防规范 | GB 50016最新版+地方消防要求 | 各专业 | — |
| 节能标准 | GB 55015最新版+地方节能要求 | 暖通/建筑 | — |
| 绿建标准 | GB/T 50378+地方绿建要求 | 绿建顾问 | 绿建等级影响概算 |
| 装配式 | 地方装配式建筑管理办法 | 结构专业 | 装配率要求 |
| 无障碍 | GB 50763最新版 | 建筑专业 | 医院无障碍要求高于一般公建 |
| 抗震 | 地方抗震设防要求 | 结构专业 | 影响结构选型 |
| ★感控 | 医院感染管理相关规范 | 建筑专业 | 洁污分流等 |
| ★辐射 | 辐射防护相关标准 | 建筑专业 | 详见5.06 |

### 8.2 ★最新地方性条文要求

> **原文强调：** "从项目立项开始，每个项目开始前需要对接的各类资料和注意事项汇总，包含国家地方的法规规范要求，最新的地方性条文要求等。"

**项目负责人须在设计开始前完成以下确认：**

- [ ] 当地最新版控规/修规要求（容积率、密度、限高等是否有变化）
- [ ] 当地消防审查的特殊要求（各地消防部门可能有额外要求）
- [ ] 当地绿建/节能的最新政策（如星级要求、奖励/处罚政策）
- [ ] 当地装配式的最新政策（装配率要求是否有变化）
- [ ] 当地人防配建要求
- [ ] 当地无障碍的特殊要求
- [ ] 当地规划部门对医院建筑的特殊要求（如外立面审查）
