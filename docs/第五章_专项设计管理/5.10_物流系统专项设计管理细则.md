# 5.10 物流系统专项设计管理细则

---

## 一、概述

医疗建筑物流系统承担院内物资（药品、标本、被服、餐食、器械、垃圾）的垂直和水平传输，直接影响运营效率和感控安全。

**核心特殊性：**
- **洁污分流**：洁净物品与污物必须分系统传输
- **时效要求高**：急诊标本、血液制品要求分钟级送达
- **全天候运行**：部分系统（气动物流）需24小时不间断
- **涉及科室广**：几乎每个护理单元、医技科室均有站点
- **与建筑布局深度耦合**：站点位置、井道预留必须在方案阶段确定

**核心原则：物流系统类型和站点布局必须在方案阶段确定，施工图阶段仅做精确定位。**

## 二、策划阶段

### 2.1 物流系统类型总览

| 系统 | 传输对象 | 传输距离 | 传输速度 | 适用场景 |
|------|---------|---------|---------|---------|
| 气动物流 | 小件（标本、药品、文件）≤5kg | 全院级（水平+垂直） | 6-8m/s | ★最常用，三甲标配 |
| 箱式物流（轨道） | 中件（药品、器械）≤20kg | 全院级 | 0.5-1.5m/s | 大批量定时配送 |
| 轨道小车 | 中件≤50kg | 全院级 | 2-5m/s | 替代气动+箱式 |
| 机器人物流（AGV/AMR） | 大件≤300kg | 同层水平 | 1-2m/s | 楼层内配送，灵活 |
| 垃圾被服物流 | 垃圾袋/被服袋 | 各层→地下集中处理 | 重力+气动 | ★大型医院推荐 |
| 餐梯/传菜梯 | 餐食 | 垂直 | 电梯速度 | 厨房→各楼层 |

### 2.2 各系统优缺点对比

| 系统 | 优点 | 缺点 | 单站点造价(万) | 推荐场景 |
|------|------|------|--------------|---------|
| 气动物流 | 速度快、全天候、适合急件 | 噪音、不能传大件、载重≤5kg | 3-5 | 标本、紧急药品、血液 |
| 箱式物流（滚轴式） | 载重大、可分时配送 | 速度慢、占空间大、留洞大 | 8-15 | 药品、器械定时配送 |
| 箱式物流（皮带式） | 载重适中、故障率低 | 留洞1600×1600mm | 6-10 | 同上 |
| 轨道小车 | 速度较快、载重居中 | 造价高、轨道占吊顶空间 | 10-20 | 替代气动+箱式组合 |
| 机器人AGV/AMR | 灵活、可扩展、不需井道 | 依赖电梯、需充电、走廊宽度要求 | 15-30/台 | 大楼层、门诊配送 |
| 垃圾被服 | 减少人工搬运、感控好 | 投资大、需专用井道 | 20-40/投口 | ≥500床医院 |

### 2.3 站点数量估算

**气动物流：**
- 每个护理单元2个站点（护士站1个、治疗室1个）
- 每个手术室区域2-3个站点
- 检验科4-8个站点（按科室分）
- 急诊2-3个站点
- 药房2-4个站点
- 血库1-2个站点
- 参考：500床医院约60-80个气动站点

**箱式物流：**
- 每个护理单元1个站点
- 药房/供应室各1-2个站点
- 参考：500床医院约20-30个箱式站点

### 2.4 需院方确认的事项

- [ ] 配置哪几种物流系统（气动+箱式+垃圾被服是常见组合）
- [ ] 各系统站点数量和分布
- [ ] 是否需要检修口朝向和物流开门方向指定（★影响平面布局）
- [ ] 箱式物流传输方式选择（滚轴式/皮带式）
- [ ] 气动物流传输方案（并联/串联）
- [ ] 机器人物流是否配置
- [ ] 各系统运营管理模式（谁操作、谁维护）
- [ ] ★请行业专家和科室主任开专题会审核确认

### 2.5 造价参考（整体系统）

| 系统 | 500床医院参考造价 | 说明 |
|------|-----------------|------|
| 气动物流（60站） | 200-350万 | 含主机、管道、站点 |
| 箱式物流（20站） | 200-400万 | 滚轴式较贵 |
| 轨道小车（30站） | 400-700万 | 高端方案 |
| 垃圾被服（15投口） | 300-500万 | 含地下收集站 |
| 机器人（5台） | 100-200万 | 含调度系统 |

## 三、初步设计阶段

### 3.1 建筑专业

#### 3.1.1 站点布局原则

| 系统 | 站点位置要求 | 空间预留 |
|------|------------|---------|
| 气动站点 | 嵌墙安装，距地面800-1000mm | 墙面预留洞600×400×300mm（深） |
| 箱式站点 | 落地安装，靠近走廊 | 占地约1.2m×0.8m，高度1.8m |
| 垃圾投口 | 各层污物间内 | 洞口约600×600mm |
| 机器人充电站 | 走廊尽端或后勤区 | 占地约1.5m×1.0m |

#### 3.1.2 井道和管道预留

| 系统 | 垂直通道要求 | 水平通道要求 |
|------|------------|------------|
| 气动物流 | 管道穿楼板预留Φ160-200mm | 吊顶内走管，需预留空间≥300mm |
| 箱式物流（滚轴式） | 井道≥1200×1200mm | 水平轨道在吊顶内或走廊上方 |
| 箱式物流（皮带式） | ★留洞1600×1600mm | 同上 |
| 垃圾被服 | 专用井道≥800×800mm | 地下层水平输送至收集站 |
| 轨道小车 | 井道≥800×800mm | 吊顶内轨道，需≥400mm净高 |

> **★ 经验教训：** 箱式物流皮带式留洞1600×1600mm，在楼板上开如此大的洞对结构影响显著，须在方案阶段就与结构专业沟通，不能施工图阶段才提出。

#### 3.1.3 ★层高影响

| 系统 | 对吊顶内空间的影响 | 说明 |
|------|------------------|------|
| 气动管道 | 占用约200mm（管径160mm+支架） | 影响较小 |
| 箱式轨道 | 占用约500-800mm | ★对走廊吊顶净高影响大 |
| 轨道小车 | 占用约400-600mm | 与暖通风管抢空间 |
| 垃圾被服 | 不占吊顶空间（走专用井道） | — |

> **★ 管线综合重点：** 物流管道/轨道与暖通风管、强弱电桥架、喷淋管道在吊顶内"抢空间"，必须在施工图阶段做BIM综合排布。

#### 3.1.4 与电梯的关系

- 机器人物流依赖电梯运行，须确认哪些电梯配置机器人对接接口
- 箱式/气动物流有独立垂直通道，不占用电梯
- ★垃圾被服物流的垂直井道不得与电梯井道合并
- 物流设备间位置宜靠近电梯核心筒（便于管道布置）

### 3.2 结构专业

| 配合事项 | 要求 |
|---------|------|
| 气动管道穿楼板 | 每处预留Φ200mm套管，对结构影响小 |
| 箱式物流井道 | ★1200×1200或1600×1600开洞，须在结构设计时同步考虑 |
| 轨道小车井道 | 800×800开洞+吊挂轨道荷载 |
| 垃圾被服井道 | 800×800专用井道，壁厚≥200mm |
| 物流设备间 | 主机基础荷载约5-10kN/m² |
| ★留洞通用性原则 | 结构留洞应具有通用性，可满足后续不同厂家的招标需求 |
| 振动 | 箱式物流运行有振动，站点附近避免设精密仪器房间 |

### 3.3 电气专业

| 项目 | 要求 |
|------|------|
| 气动主机供电 | 三相380V，功率15-30kW/台 |
| 箱式物流供电 | 三相380V，功率20-50kW |
| 各站点供电 | 单相220V，每个站点预留1个回路 |
| UPS | 气动物流主机建议配UPS（急诊标本传输不能中断）|
| 弱电接口 | 各站点需网络接口（与HIS/物流管理系统通信）|
| 消防联动 | ★气动物流在火灾时自动停机（防止火烟通过管道传播）|

### 3.4 暖通专业

| 项目 | 要求 |
|------|------|
| 气动主机房 | 通风散热，主机产热量约5-15kW |
| 垃圾被服收集站 | ★负压通风、除臭处理 |
| 噪音控制 | 气动物流管道经过病房区域时须设消音器 |
| ★消防 | 垃圾被服井道在每层楼板处设防火阀（井道为竖向风道）|

### 3.5 给排水专业

| 项目 | 要求 |
|------|------|
| 垃圾被服收集站 | 设给水（冲洗用）和排水（地面排水） |
| 被服收集站 | 设给水（预洗用，如有） |
| 气动站点 | 无给排水需求 |

## 四、施工图设计阶段

### 4.1 施工图前须确认的前置条件

| 序号 | 确认事项 | 确认方 | 时间节点 |
|------|---------|-------|---------|
| 1 | 物流系统配合单位确定 | 甲方/总包/设计院 | ★方案阶段就需确定 |
| 2 | 各系统站点布局确认 | 院方+设计院+厂家 | 施工图开始前 |
| 3 | 垂直通道留洞位置确认 | 结构+建筑+厂家 | 结构施工图出图前 |
| 4 | 设备间位置及面积确认 | 建筑+厂家 | 施工图阶段 |
| 5 | 是否需要装修设备间 | 甲方 | 施工图阶段 |

### 4.2 各专业施工图要点

**建筑专业：**
- 平面图中标注所有站点位置、编号
- 剖面中标注井道位置和各层留洞
- ★站点检修口方向和开门方向须标注（影响相邻房间布局）
- 气动管道水平走向须在吊顶综合图中体现
- 设备间平面布局图

**结构专业：**
- 各层留洞位置、尺寸标注在结构图中
- 留洞周边加强筋做法
- 设备基础
- ★所有留洞必须在结构图中表达，不能只在物流图中表达

**电气专业：**
- 各站点配电回路
- 设备间配电箱
- 网络接口预留
- 消防联动方案

### 4.3 常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 方案阶段未确定物流系统 | 施工图阶段加站点导致平面改动 | ★方案阶段即确定系统类型和站点数量 |
| 结构留洞未预留或位置不对 | 后开洞影响结构安全 | 物流厂家确定后2周内必须提留洞图 |
| 气动管道穿防火分区未设防火阀 | 消防验收不通过 | 每穿一处防火分区/楼板均设防火阀 |
| 箱式物流轨道与暖通风管打架 | 吊顶标高被迫降低 | BIM管线综合 |
| 垃圾被服井道穿楼板封堵未做 | 烟气通过井道蔓延 | ★封堵做法应在土建图中表达，非物流图中 |
| 气动物流噪音扰民 | 病房区患者投诉 | 管道经过病房区段设消音器，站点避开床头 |
| 站点检修口方向错误 | 检修时须拆除相邻设施 | 检修口朝走廊侧，不朝房间内侧 |

## 五、厂家配合

### 5.1 配合单位确定原则

> **原文强调：** "箱式物流一般由设计院向建设方或总包方提出确定配合单位的要求，都不确定的话由设计院根据项目特点找相关厂家配合。"

### 5.2 配合时序

```
┌─────────────────────────────────────────────────────────┐
│ 阶段            │ 厂家须提供              │ 设计院动作     │
├─────────────────────────────────────────────────────────┤
│ 方案阶段        │ 系统选型建议+初步造价    │ 确定系统类型   │
│                 │ 站点初步布局建议         │ 确定站点数量   │
├─────────────────────────────────────────────────────────┤
│ 初设阶段        │ 井道留洞图              │ 纳入结构设计   │
│                 │ 设备间条件              │ 纳入建筑布局   │
├─────────────────────────────────────────────────────────┤
│ 施工图阶段      │ 详细施工图              │ 审核+碰撞检查  │
│                 │ 配电需求                │ 纳入电气设计   │
├─────────────────────────────────────────────────────────┤
│ 施工阶段        │ 现场安装技术指导         │ 配合验收       │
└─────────────────────────────────────────────────────────┘
```

### 5.3 ★厂家技术配合原则

> **原文强调：** "物流原则是结构留洞具有通用性，站点设置由设计院确定，物流厂家根据项目特点造价等提供建议。"
> "审核由建设方审核确认站点数量及物流传输方式的类型，比如皮带式还是滚轴式，请行业专家和科室主任等开专题会议审核确定。"

## 六、施工阶段配合

| 节点 | 配合内容 | 负责方 |
|------|---------|-------|
| 结构施工 | 留洞位置复核，预埋套管 | 结构+物流厂家 |
| 管道安装 | 气动管道、轨道安装 | 物流厂家 |
| 穿楼板封堵 | ★上下楼层之间穿楼板的预留洞周边封堵 | 施工单位（非物流厂家） |
| 气动屋顶开洞 | ★气动垃圾物流屋顶开洞要提前确认并预留套管 | 建筑+结构 |
| 系统调试 | 全系统联调 | 物流厂家+电气+弱电 |
| 消防联动 | 火灾信号→物流停机 | 电气+消防+物流厂家 |

> **★ 原文强调：** "上下楼层之间，穿楼板的预留洞周边封堵，做法应在土建图纸中表达（设计说明或局部详图），而非在垃圾被服图纸中表达。施工界面问题。"

## 七、验收配合

### 7.1 验收内容

| 验收项 | 标准 | 检测方法 |
|-------|------|---------|
| 传输速度 | 气动≥5m/s，箱式≥0.5m/s | 实测 |
| 传输可靠性 | 连续运行100次无故障 | 实测 |
| 噪音 | 站点处≤55dB(A)，管道经过病房段≤45dB(A) | 声级计 |
| 消防联动 | 火灾信号发出后系统停止运行 | 功能测试 |
| 防火阀 | 动作可靠、密封性良好 | 逐个测试 |
| 楼板封堵 | 耐火完整性满足要求 | 外观检查+耐火试验报告 |
| 站点功能 | 发送/接收/存储/报警功能正常 | 逐站测试 |

## 八、各系统详细技术参数

### 8.1 气动物流

| 参数 | 数值 |
|------|------|
| 传输管径 | Φ160mm（标准）/ Φ200mm（大口径） |
| 传输载体 | 专用传输瓶，长约400mm |
| 载重 | ≤5kg |
| 速度 | 6-8m/s |
| 传输方案 | **并联**：每站独立管道，造价高但互不干扰；**串联**：共享管道，造价低但效率低 |
| 噪音 | 站点处约55-65dB(A) |
| 主机 | 真空泵+正压泵，功率15-30kW |
| 代表厂家 | Swisslog、Sumetzberger、TransLogic、国内：新松、远图 |

> **★ 建议：** 三甲医院推荐并联方案，站点数>30个时串联效率显著下降。

### 8.2 箱式物流（滚轴式）

| 参数 | 数值 |
|------|------|
| 传输箱尺寸 | 约600×400×400mm |
| 载重 | ≤20kg |
| 速度 | 0.5-1.0m/s |
| 留洞尺寸 | 1200×1200mm（垂直通道） |
| 代表厂家 | Swisslog、德马泰克、昆船 |
| 造价 | 8-15万/站 |
| ★消防问题 | 传输箱为可燃物，井道内须设感烟探测+自动灭火 |

### 8.3 箱式物流（皮带式）

| 参数 | 数值 |
|------|------|
| 传输箱尺寸 | 约600×400×400mm |
| 载重 | ≤15kg |
| 速度 | 0.5-1.5m/s |
| ★留洞尺寸 | **1600×1600mm** |
| 代表厂家 | Hänel、国内多家 |
| 造价 | 6-10万/站 |

### 8.4 轨道小车

| 参数 | 数值 |
|------|------|
| 载重 | ≤50kg |
| 速度 | 2-5m/s |
| 轨道截面 | 约300×200mm（吊装于吊顶内） |
| 垂直通道 | 专用提升机，井道≥800×800mm |
| 代表厂家 | Swisslog（TransCar）、新松 |
| 造价 | 10-20万/站 |
| 优势 | 速度快、载重大，可替代气动+箱式组合 |

### 8.5 机器人物流（AGV/AMR）

| 参数 | 数值 |
|------|------|
| 载重 | ≤300kg |
| 速度 | 1-2m/s |
| 导航方式 | 激光SLAM/视觉导航（AMR），磁条/二维码（AGV）|
| 走廊宽度要求 | ≥2.0m（双向通行≥2.4m） |
| 充电 | 自动回充电站，充电时间2-4h，续航8-12h |
| 电梯对接 | 须与电梯厂家协调API接口 |
| 代表厂家 | 诺亚、极智嘉、擎朗、新松 |
| 造价 | 15-30万/台（含调度系统） |
| ★注意 | 依赖电梯跨层运输，高峰期可能影响电梯效率 |

### 8.6 垃圾被服物流

| 参数 | 数值 |
|------|------|
| 投口尺寸 | 约600×600mm |
| 传输方式 | 重力+气动推送 |
| 垂直井道 | 专用竖井，≥800×800mm |
| 地下收集站面积 | 30-60m² |
| 代表厂家 | 新华集团、吉之美 |
| 造价 | 20-40万/投口 |
| ★消防 | 井道内每层设防火阀；投口门常闭、耐火极限≥1h |
