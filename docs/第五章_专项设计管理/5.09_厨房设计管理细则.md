# 5.09 厨房设计管理细则

---

## 一、概述

医院厨房类型：

| 类型 | 服务对象 | 特殊要求 |
|------|---------|---------|
| 营养餐厨房 | 住院患者 | ★须满足治疗膳食需求（糖尿病餐/肾病餐/术后流食等）|
| 职工餐厅 | 医护人员 | 就餐时间集中、翻台率高 |
| 中心厨房 | 全院 | 面积大、设备多、配送系统 |

---

## 二、内部配合

### 2.1 建筑专业

> **★ 原文强调：** "布局由建筑专业划定大致的厨房区域后，由专业分包单位深化平面布局，设计院审核后提交甲方确认平面布局，提示甲方最好有厨房餐厅运营公司参与设计。"

| 配合事项 | 要求 |
|---------|------|
| 布局流程 | 建筑划定区域 → 分包深化 → 设计院审核 → 甲方确认 |
| ★运营公司参与 | 提示甲方最好有运营公司参与设计 |
| 明沟 | ★明沟应在建筑图纸中表达，由分包单位提供位置 |
| 层高 | 净高≥3.0m（排烟罩+排风管道占空间）|
| 防滑地面 | 厨房地面防滑等级≥R12 |
| 隔油池 | 室外设隔油池，位置在建筑总图中标注 |

### 2.2 结构专业

| 配合事项 | 要求 |
|---------|------|
| 荷载 | 厨房设备区≥6kN/m²（大型蒸柜/洗碗机/冷库）|
| 降板 | 厨房区域降板50-80mm（明沟+防水+地面找坡）|
| 冷库基础 | 大型冷库须独立基础 |

### 2.3 给排水专业

| 配合事项 | 要求 |
|---------|------|
| 给水 | 用水量大，主管径≥DN50 |
| 热水 | 蒸汽或热水锅炉供热水，管道预留 |
| 排水 | 明沟排水+地漏排水，接隔油池后排市政 |
| 隔油 | ★必须设隔油池，容积按日排水量计算 |

### 2.4 暖通专业

| 配合事项 | 要求 |
|---------|------|
| 排烟罩 | 灶台上方设排烟罩，排风量大 |
| 补风 | 排风量大，须同步补风（否则负压严重）|
| 空调 | 厨房操作区不设回风（油烟），可设岗位送风 |

### 2.5 电气专业

| 配合事项 | 要求 |
|---------|------|
| 供电 | 大功率设备（蒸柜/洗碗机/冷库）专用回路 |
| 燃气报警 | 如使用天然气，须设可燃气体探测器+联动切断阀 |
| 照明 | ≥200lx，灯具须防水防油 |

## 三、分包方管控

| 阶段 | 分包须提交 | 设计院审核 |
|------|----------|----------|
| 方案 | 厨房平面布局+设备清单 | 与建筑平面协调、流线合理性 |
| 施工图 | 设备基础图、水电点位图 | 与土建各专业条件一致 |

### 分包须提供的图纸

| 图纸 | 内容 |
|------|------|
| 装饰图纸 | 墙/地/顶做法 |
| 电气图纸 | 各设备配电点位 |
| 暖通图纸 | 排烟罩位置和排风管路 |
| 二次水点位 | 各设备给排水点位 |

---

## 四、常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 无运营公司参与设计 | 建成后流线不合理、设备放不下 | ★提示甲方引入运营公司 |
| 明沟未在建筑图中表达 | 施工单位不知道做明沟 | 建筑图纸中明确明沟位置和做法 |
| 隔油池容积不够 | 排水堵塞、异味 | 按日排水量×1.5倍计算容积 |
| 排风量不足 | 厨房油烟弥漫 | 灶台排风量≥2500m³/h/灶眼 |
