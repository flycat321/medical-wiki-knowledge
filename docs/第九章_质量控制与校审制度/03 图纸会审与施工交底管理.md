### 3.1 图纸会审

| 阶段 | 参加方 | 内容 | 输出 |
|------|-------|------|------|
| 内部会审 | 设计院各专业 | 跨专业校核、管线综合、错漏检查 | 内部会审纪要 |
| 外部会审 | 设计院+施工单位+监理 | 施工可行性审查、疑问解答 | 外部会审纪要 |
| 审图会审 | 审图机构 | 规范符合性、强条检查 | 审图意见 |

### 3.2 审图意见处理

```
审图意见处理流程：

  收到审图意见
       │
       ▼
  各专业逐条认领并回复
       │
  ├── 同意修改 → 修改图纸 → 标注修改内容 → 回复"已改"
  │
  └── 不同意 → 提出理由和依据 → 与审图机构沟通 → 达成一致
       │
       ▼
  汇总回复函 → 项目负责人审核签字 → 提交审图机构
       │
       ▼
  取得审查合格书
```
