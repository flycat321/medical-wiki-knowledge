# 5.12 机械停车库专项设计管理细则

---

## 一、概述

医院停车难是普遍问题。机械停车库可在有限空间内增加车位密度，但医院场景有特殊限制。

---

## 二、策划阶段

### 2.1 类型选择

| 类型 | 适用场景 | 车位密度 | 平均取车时间 | 单车位造价(万) |
|------|---------|---------|------------|--------------|
| 升降横移 | 地上/地下，改造方便 | 中 | 60-120s | 2-4 |
| 垂直循环 | 地上，占地小 | 高 | 90-150s | 3-5 |
| 平面移动 | 地下，大规模 | 高 | 60-90s | 4-8 |
| 巷道堆垛 | 地下，大规模 | 最高 | 90-120s | 5-10 |
| 垂直升降（塔式） | 地上，极小占地 | 最高 | 120-180s | 8-15 |

### 2.2 ★医院停车特殊限制

| 限制 | 说明 |
|------|------|
| ★急诊区不得用机械车位 | 急救车辆需即停即走 |
| ★无障碍车位不得用机械车位 | 轮椅使用者无法操作 |
| ★门诊高峰效率 | 门诊区域机械车位取车等待时间须评估 |
| 值班/急诊人员车位 | 须设常规车位，24h可用 |

### 2.3 需甲方确认

- [ ] 机械车位数量和类型
- [ ] 设备品牌偏好
- [ ] 是否地上/地下
- [ ] 运营维保方案

---

## 三、设计要点

### 3.1 建筑专业

| 要点 | 要求 |
|------|------|
| 车位尺寸 | 按设备型号确定（通常宽≥2.5m、长≥5.3m、高≥1.55m/层） |
| 通道宽度 | 出入口处≥6.0m（双向） |
| 层高 | 升降横移：净高≥2.0m/层；塔式：总高度大 |
| ★与常规车位衔接 | 机械区域与常规停车区域之间设过渡区 |

### 3.2 结构专业

| 要点 | 要求 |
|------|------|
| 荷载 | 按设备厂家提供的荷载布置图 |
| 基坑 | 升降横移下沉式：基坑深度≥1.8m |
| 振动 | 运行振动对相邻区域的影响评估 |

### 3.3 电气专业

| 要点 | 要求 |
|------|------|
| 供电 | 三相380V，每组设备15-30kW |
| 消防联动 | 火灾信号→设备停止运行→车辆降至最低位 |
| 照明 | 出入口处≥100lx |

### 3.4 暖通专业

| 要点 | 要求 |
|------|------|
| 通风 | 机械车库通风量≥6次/h（尾气排放）|
| 排烟 | 按地下车库排烟标准 |

---

## 四、厂家配合

> **★ 原文强调：** "不同设备厂家方案对比，原则是通用性，满足后续不同厂家招标需求。"

| 原则 | 说明 |
|------|------|
| 通用性 | 结构留洞和基坑尺寸按最大包络设计 |
| 可比性 | 招标时至少3个品牌可投标 |
| 标准化 | 车位尺寸按国标最大值预留 |
