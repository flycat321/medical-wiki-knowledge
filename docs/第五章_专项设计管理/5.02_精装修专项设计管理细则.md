# 5.02 精装修专项设计管理细则（含固定家具、医用家具）

---

## 一、概述

医疗建筑精装修区别于普通公建的核心差异：

- **感控优先**：材料须抗菌、耐擦洗、无死角、易清洁
- **功能集成度高**：护士站、诊室背景墙集成大量医疗设备面板
- **多专项交叉**：与净化、防护、标识、智能化均有界面
- **分区标准差异大**：大厅/诊疗区/住院区/后勤区装修标准和造价完全不同
- **医用家具特殊**：护士站、诊桌、治疗台等与医疗流程强相关

## 二、策划阶段

### 2.1 装修分级与造价参考

| 区域 | 装修定位 | 参考单方造价(元/m²) | 说明 |
|------|---------|-------------------|------|
| 门诊大厅/入口 | 高标准 | 2,500-4,500 | 品牌形象区 |
| 诊室/治疗室 | 中标准 | 1,200-2,000 | 功能为主 |
| 病房/护理单元 | 中标准 | 1,500-2,500 | 含床头背景墙系统 |
| 手术部/ICU | 净化标准（单列） | — | 由净化分包负责 |
| 走廊/电梯厅 | 中标准 | 1,000-1,800 | 含防撞/扶手 |
| 餐厅 | 中标准 | 1,500-2,500 | — |
| 行政办公 | 一般标准 | 800-1,500 | — |
| 后勤/设备用房 | 基本装修 | 300-600 | 仅地面+墙面涂料 |
| 报告厅/会议中心 | 高标准 | 3,000-5,000 | 含声学处理 |

### 2.2 合同附件——开始施工图前项目负责人须明确6项事项

> **★ 以下6项必须在精装施工图开始前以书面形式确认，作为精装分包合同附件：**

| 序号 | 事项 | 说明 |
|------|------|------|
| 1 | 设计定位 | 甲方和设计院对装修标准、风格、色彩的要求，形成交底文件并开会交底 |
| 2 | 造价控制要求 | 分区域限额，精装总造价控制目标 |
| 3 | 所有墙顶地材料、规格、收边节点处理 | 须附相关资料清单说明 |
| 4 | 与净化/防护/实验室专项之间的界面划分 | ★关键：哪些区域精装做，哪些净化做 |
| 5 | 固定家具和活动家具的界限范围 | 护士站台面算固定还是活动？床头柜呢？ |
| 6 | 消防专项是否包含 | 喷淋/烟感/应急灯由精装协调还是消防专业独立 |

> **★ 交底会意见汇总后，形成文件，交付到项目管理文件夹并扫描汇总至电子版文件管理库。**

## 三、内部配合（土建各专业）

### 3.1 建筑专业

| 配合事项 | 要求 |
|---------|------|
| ★大厅防火分区 | 大厅防火分区的划分应与装饰方案协商，避免出现需做大量甲级防火玻璃的情况 |
| 吊顶标高 | 管线综合后确定各区域吊顶标高，精装按此标高设计 |
| 墙面龙骨 | 石材/大板干挂区域须建筑提供基层墙面要求 |
| 地面标高 | 不同区域地面材料厚度不同，须协调标高过渡 |
| 门洞 | 精装门套尺寸须与建筑门洞匹配 |

### 3.2 结构专业

| 配合事项 | 要求 |
|---------|------|
| 石材干挂 | 大面积石材干挂须结构验算墙面/吊顶荷载 |
| 吊顶荷载 | 集成吊顶（灯具+风口+设备）荷载提交结构 |
| 地面降板 | 大厅石材地面厚度较大，须确认是否需要降板 |

### 3.3 给排水专业

> **★ 原文强调：** "土建排布卫生间以及布置家具时，要把给排水管道落入图纸后排布，否则会出现给排水管道影响卫生间洗手盆的布局以及开门的门垛尺寸。包立管应在土建图中体现。"

| 配合事项 | 要求 |
|---------|------|
| 卫生间排布 | ★先落排水管位置，再布置洁具和家具 |
| 包立管 | 立管包管位置和尺寸应在土建图中体现 |
| 洗手池 | 感应龙头供电预留（电气配合） |

### 3.4 暖通专业

| 配合事项 | 要求 |
|---------|------|
| 风口位置 | 送风口/回风口位置须与吊顶方案协调 |
| 新风口 | 新风口百叶位置不得影响装饰立面 |
| 风管检修口 | 吊顶设计须预留风管检修口位置 |

### 3.5 电气专业

| 配合事项 | 要求 |
|---------|------|
| 开关插座 | 精装出定位图后电气按图施工 |
| 应急照明 | ★精装须提取一次机电中应急照明相关内容配合 |
| 紫外灯 | 位置及定位、连线 |
| LED/液晶屏 | 所有屏幕的电气连线及开关位置 |
| 灯具 | 精装灯具选型须与电气确认功率和配电回路 |

## 四、方案设计内容与深度

### 4.1 设计方案册须包含

- 设计说明（定位、风格、色彩体系）
- 平面布局（含家具布置）
- 吊顶设计（★须考虑：照明、风口、喷淋等消防点位综合排布）
- 地面铺装图
- 立面展开图（重点空间）
- 材料意向板
- 效果图（重点空间）

### 4.2 各功能区设计要点

#### ★大厅

| 要点 | 说明 |
|------|------|
| 水炮vs喷淋 | ★高度<18m的大厅可做水炮代替喷淋，取消吊顶喷淋点位，给吊顶设计预留条件，且更美观 |
| LED/液晶屏 | 大位置、大小；候诊/取药门口屏幕位置大小须标注 |
| 导诊台 | 位置醒目，与标识系统联动 |
| 无障碍 | 地面防滑等级、盲道、无障碍标识 |
| 防火分区 | ★与装饰方案协商防火分区划分线位置 |

#### ★诊室

| 要点 | 说明 |
|------|------|
| 诊桌布局 | 医生面对门口（安全感）、患者椅在侧面 |
| 隐私 | 诊室门设观察窗但须遮挡患者检查区 |
| 信息化设备 | 电脑、打印机、读卡器位置预留 |

#### ★走廊

| 要点 | 说明 |
|------|------|
| 防撞扶手 | 距地面850-900mm，转角处做圆弧过渡 |
| 防撞护墙板 | 距地面0-350mm区域设防撞护板（推车/轮椅撞击区）|
| 无障碍 | 净宽≥2.4m（病床推行），设盲道或触觉引导 |

#### ★病房

| 要点 | 说明 |
|------|------|
| 床头背景墙 | ★集成系统：医用气体面板、呼叫按钮、照明开关、阅读灯、USB充电口、信息面板 |
| 卫生间 | 无障碍洗手间、防滑地面、扶手、呼叫按钮 |
| 隐私帘轨道 | 吊顶预埋 |

#### ★护理单元走廊

| 要点 | 说明 |
|------|------|
| 护士站 | ★设计重点：台面高度（坐姿工作面750mm/站姿接待面1100mm），集成系统（电脑、呼叫主机、气动站点、物品存放）|
| 治疗室/处置室 | 台面、吊柜、洗手池布局 |
| 走廊宽度 | ≥2.7m（双向病床通行） |

#### 餐厅

| 要点 | 说明 |
|------|------|
| 取餐区 | 动线设计、隔断材料须耐擦洗 |
| 座位 | 无障碍餐位≥5% |

### 4.3 选型表体系

精装方案须附以下选型表，经甲方确认后作为施工图和采购依据：

| 选型表 | 内容 |
|-------|------|
| 灯具选型表 | 各区域灯具型号、功率、色温、数量 |
| 物料表 | 墙面/地面/吊顶所有材料品牌、规格、色号 |
| 室内门窗选型表 | 各类门（诊室门/病房门/卫生间门/防火门）的材质、五金 |
| 洁具选型表 | 洗手盆、马桶、淋浴等品牌型号 |
| 五金选型表 | 门把手、铰链、闭门器、扶手等 |
| 报告厅座椅选型 | 品牌、面料、功能（翻写/折叠）|

## 六、医用家具专项

### 6.1 分类

| 类别 | 包含内容 | 特点 |
|------|---------|------|
| 固定医用家具 | 护士站台面、治疗台、药品柜（嵌入式）、换药台 | 与建筑/水电强相关，须在施工图中定位 |
| 活动医用家具 | 诊桌、诊椅、病床、床头柜、输液架、治疗车 | 采购为主，对建筑影响小 |
| ★定制医用家具 | 护士站整体定制、特殊科室操作台 | 须厂家出深化图，设计院审核 |

### 6.2 护士站设计要点（★核心）

| 要点 | 说明 |
|------|------|
| 台面高度 | 内侧工作面750mm（坐姿），外侧接待面1100mm（站姿） |
| 集成设备 | 电脑显示器×2-4、呼叫主机、气动物流站、打印机、扫码枪 |
| 电气需求 | 每工位至少4个插座+2个网口+1个电话口 |
| 材质 | 台面：人造石/抗菌HPL；柜体：抗菌板材 |
| 尺寸 | 每工位宽≥1200mm，柜体深度≥600mm |
| ★注意 | 护士站设计须与弱电专业（呼叫系统主机）同步确认 |

## 八、施工阶段配合

| 节点 | 配合内容 |
|------|---------|
| 样板间 | ★重要区域（病房、护士站、大厅）须先做样板间，院方确认后再批量施工 |
| 材料进场 | 核对品牌型号与选型表一致 |
| 隐蔽工程 | 吊顶内管线综合安装完成后，封板前验收 |
| 面层施工 | 检查材料拼缝、色差、平整度 |
| 家具安装 | 护士站等定制家具安装后功能性验收 |
| 收边收口 | 不同材料交接处的节点处理是品质关键 |

## 九、常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 大厅防火分区线与装饰方案冲突 | 大面积甲级防火玻璃造价高昂 | ★方案阶段即协调 |
| 卫生间管道位置未落图就布置洁具 | 管道与洗手盆/门垛冲突 | ★先落管道再布家具 |
| 精装图册打印规格不统一 | 院方领导不满 | ★济南第一医院教训：统一A3规格 |
| 出差汇报未提前测试设备 | 到现场才发现投影/电脑问题 | ★带自己电脑、提前拷贝、提前测试 |
| 精装吊顶标高与管线综合矛盾 | 实际吊顶比设计低，空间压抑 | BIM管线综合后再定吊顶标高 |
| 床头背景墙未集成医用气体面板 | 后期凿墙改管 | ★床头墙设计须与医用气体专业同步 |
| 固定家具与活动家具界面不清 | 甲方和分包互相推诿 | 合同附件中明确界限 |
