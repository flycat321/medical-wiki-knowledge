除常规校审内容外，医疗建筑设计须增加以下专项校审内容：

### 2.1 感控流线校审

| 校审内容 | 检查标准 | 校审人 |
|---------|---------|-------|
| 洁污分流 | 洁净物品和污物不共用通道/电梯 | 建筑专业 |
| 医患分流 | 医护通道和患者通道有效分离 | 建筑专业 |
| 手术部流线 | 患者/医护/器械/污物四条线分离 | 建筑专业 |
| CSSD单向流 | 去污→检包→无菌，不可逆流 | 建筑专业 |
| 负压隔离 | 三区两缓是否完整 | 建筑+暖通 |
| 太平间独立性 | 遗体运送不经过患者区域 | 建筑专业 |

### 2.2 净高校审

| 校审内容 | 检查标准 | 校审人 |
|---------|---------|-------|
| 手术室净高 | Ⅰ级≥3.0m、Ⅱ-Ⅳ级≥2.8m（吊顶下） | 建筑+暖通 |
| 走廊净高 | 管线综合后≥2.7m | 各专业联合 |
| ICU净高 | ≥2.7m | 建筑+暖通 |
| 地下室净高 | 停车区≥2.2m、设备区按需 | 建筑+结构 |
| ★吊顶内空间 | 手术层≥1200mm、普通层≥600mm | BIM管综 |

### 2.3 防护校审

| 校审内容 | 检查标准 | 校审人 |
|---------|---------|-------|
| 防护区降板 | 是否按防护预评价确定深度 | 建筑+结构 |
| 上下层关系 | 防护区上下层防护是否达标 | 建筑 |
| MRI 5高斯线 | 是否标注、范围内是否有磁性材料 | 建筑+结构 |
| 运输路径 | 是否标注、是否逐段验证 | 建筑 |
| 失超排气 | MRI是否设置 | 暖通 |

### 2.4 消防校审

| 校审内容 | 检查标准 | 校审人 |
|---------|---------|-------|
| 消防电梯 | 数量/位置/前室面积/排水 | 建筑+给排水 |
| 防火分区 | ★大厅分区线是否与装饰协调 | 建筑 |
| 疏散距离 | 是否满足规范 | 建筑 |
| 消防联动 | 电梯迫降/风阀关闭/卷帘下降/物流停机 | 电气 |
| 气体灭火 | 机房/配电室/档案室 | 给排水+暖通 |
