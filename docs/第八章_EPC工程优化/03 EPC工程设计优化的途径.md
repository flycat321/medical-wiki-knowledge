### 3.1 设计阶段优化

| 优化方向 | 具体措施 | 潜在节约 |
|---------|---------|---------|
| 结构优化 | 柱网调整、梁截面优化、基础方案比选 | 结构造价5-15% |
| 体型优化 | 减少凹凸、控制体型系数 | 围护结构造价10-20% |
| 系统简化 | 机电系统方案比选（多联机vs冷水机组等） | 机电造价5-10% |
| 标准化设计 | 护理单元/诊室/病房标准化 | 减少设计工时+施工效率提升 |
| 材料替换 | 在满足功能前提下选择性价比更高的材料 | 视替换范围 |
| ★净化降级 | 手术室等级合理配比 | 净化造价20-30% |
| ★智能化精简 | 非必须系统列为预留/分期 | 弱电造价20-40% |

### 3.2 施工阶段优化（可施工性审查）

| 优化方向 | 具体措施 | 说明 |
|---------|---------|------|
| 钢筋优化 | 与施工单位协商钢筋规格归并（减少规格种类） | 减少施工损耗 |
| 模板优化 | 标准化柱/梁截面，减少模板种类 | 加快周转 |
| 预制率优化 | 合理确定预制构件范围和拆分方式 | 平衡造价和进度 |
| 管线综合 | BIM碰撞检查→提前解决碰撞 | 减少现场返工 |
| 施工分区 | 设计配合施工流水分区出图 | 配合施工节奏 |

### 3.3 采购阶段优化

| 优化方向 | 具体措施 | 说明 |
|---------|---------|------|
| 设备选型 | 技术参数+造价+维保成本综合比选 | 不唯价格论 |
| 品牌比选 | 至少3个品牌技术经济对比 | 设计院提供技术评估 |
| ★通用性设计 | 设计预留条件具有通用性，不锁定单一品牌 | 便于竞争性招标 |
| 集中采购 | 同类设备/材料集中招标 | 量大价优 |
