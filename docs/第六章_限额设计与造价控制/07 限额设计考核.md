### 7.1 各阶段考核节点

| 阶段 | 考核内容 | 考核标准 | 考核责任人 |
|------|---------|---------|----------|
| 初设概算 | 概算不超可研估算 | 超10%须调整 | 项目负责人 |
| 施工图预算 | 预算不超初设概算 | 超5%须调整 | 各专业负责人 |
| 施工阶段 | 变更引起的造价增减 | 净增不超预备费 | 项目负责人 |
| 竣工结算 | 结算不超预算（合理变更除外） | — | 项目负责人 |

### 7.2 超限处理

```
超限处理流程：

  发现超限（概算核对/预算核对）
       │
       ▼
  分析超限原因（哪个专业/哪个分项超标）
       │
       ▼
  制定优化方案（至少2个方案，含技术经济对比）
       │
       ▼
  项目负责人审核优化方案
       │
       ├── 可以在设计层面消化 → 执行优化 → 重新核算
       │
       └── 无法在设计层面消化 → 报甲方调整控制目标
```
