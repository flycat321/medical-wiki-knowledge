## 四、品牌选用原则

| 原则 | 说明 |
|------|------|
| 功能匹配 | 不追求最贵，追求最适合 |
| 售后保障 | 医院设备24h运行，售后响应时间≤4h |
| 配件通用 | 优先选择配件通用性强的品牌 |
| 当地经销 | 优先选择当地有经销商/办事处的品牌 |
| ★同类统一 | 同一项目中同类设备尽量统一品牌（减少维保复杂度） |
| 概算匹配 | 品牌档位须与限额设计目标匹配 |
