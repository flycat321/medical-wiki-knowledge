# 5.04 净化专项设计管理细则

---

## 一、概述

医疗建筑净化系统区别于工业洁净室，核心差异在于：

- **服务对象是人**：温湿度、噪声、照度均有更严格的舒适性要求
- **感控要求高**：正负压控制、气流组织直接关系院感安全
- **运营时间长**：手术室每天运行12-16小时，供应室几乎24小时
- **涉及科室多**：一个三甲医院可能有10+个独立净化区域
- **与土建界面复杂**：净化装修、一次机电、二次机电三方交叉

**核心原则：净化区域的层高、气流方向、压差梯度必须在方案阶段确定，施工图阶段不应再调整净化区边界。**

### 涉及的功能区域

| 区域 | 洁净等级 | 特殊要求 |
|------|---------|---------|
| 手术室（Ⅰ级/百级） | ISO 5 局部 + ISO 7 周边 | 层流送风天花、正压控制 |
| 手术室（Ⅱ级/千级） | ISO 6 | 正压控制 |
| 手术室（Ⅲ级/万级） | ISO 7 | 正压控制 |
| 手术室（Ⅳ级/三十万级） | ISO 8 | 正压控制 |
| ICU | ISO 7-8 | 正压（普通）/负压（隔离） |
| NICU（新生儿） | ISO 7 | 正压、温湿度精确控制 |
| 产房（洁净产房） | ISO 7-8 | 正压 |
| 供应室 CSSD | ISO 7-8（无菌区） | ★单向流：污→洁→无菌，不可逆 |
| 检验科微生物室 | ISO 7-8 | 负压 |

### 2.2 ★手术室等级划分

| 等级 | 洁净级别 | 适用手术类型 | 层流天花面积 |
|------|---------|------------|------------|
| Ⅰ级 | 百级（ISO 5） | 关节置换、器官移植、心脏外科、神经外科 | ≥2.4m×2.6m |
| Ⅱ级 | 千级（ISO 6） | 胸外科、整形外科、泌尿外科、骨科（非关节置换） | — |
| Ⅲ级 | 万级（ISO 7） | 普外科、妇产科、五官科 | — |
| Ⅳ级 | 三十万级（ISO 8） | 感染类手术、急诊手术 | — |

> **★ 经验提示：** 院方通常希望所有手术室都做Ⅰ级，但造价差异巨大（Ⅰ级约8000-12000元/m²，Ⅲ级约4000-6000元/m²）。建议按手术量和学科规划合理配比，一般Ⅰ级占15-20%，Ⅱ级占20-30%，Ⅲ级占40-50%，Ⅳ级占10%。

### 2.3 ★ICU正压/负压选择

| 类型 | 压差方向 | 适用场景 | 设计要点 |
|------|---------|---------|---------|
| 正压ICU | 室内→走廊 | 普通ICU、心外ICU、NICU | 防止走廊污染空气进入 |
| 负压隔离间 | 走廊→缓冲→室内 | 传染病隔离、多重耐药菌隔离 | ★三区两缓（清洁区→半污染区→污染区，两道缓冲间）|
| 正负压可切换 | 灵活切换 | "平疫结合"设计要求 | ★造价增加约30%，需设切换控制面板 |

> **★ 重要提示：** 2020年后新建三甲医院ICU普遍要求配置不少于总床位10%的负压隔离间，部分地方要求更高。规划阶段须确认当地卫健委要求。

### 2.4 ★供应室CSSD流线

```
CSSD"单向流"原则（不可逆流）：

  污染物品 ──→ 去污区 ──→ 检查包装灭菌区 ──→ 无菌物品存放区 ──→ 发放
     │          (负压)         (正压)              (最高正压)
     │
     └── 回收窗口（与各科室连接，通常经污物电梯）

  ★ 人流同样单向：工作人员从清洁侧进入，污染侧不得返回
  ★ 去污区与检查包装区之间设传递窗/双扉互锁传递柜
  ★ 三区之间不得有直通门
```

### 2.5 需院方确认的事项

- [ ] 手术室数量及等级分配
- [ ] ICU负压隔离间数量及"平疫结合"要求
- [ ] 供应室处理量（每日灭菌包数量，决定设备选型和面积）
- [ ] PCR实验室是否需要（及检测通量）
- [ ] PIVAS日配液量（决定操作台数量和面积）
- [ ] 生殖中心是否设置（对VOC控制要求极高，影响装修材料选择）
- [ ] 负压隔离病房数量及分布楼层
- [ ] 净化系统品牌偏好

## 三、初步设计阶段

### 3.1 建筑专业

#### 3.1.1 ★层高控制（核心问题）

净化区域层高是医疗建筑设计中最容易出问题的环节之一。

| 区域 | 吊顶内净高 | 室内净高要求 | 建议结构层高 | 说明 |
|------|----------|------------|------------|------|
| 手术室 | ≥1200mm | ≥2800mm（Ⅰ级≥3000mm） | ≥4500mm | 吊顶内含送风静压箱+管线 |
| 手术部走廊 | ≥800mm | ≥2600mm | 同手术室层 | — |
| ICU | ≥1000mm | ≥2700mm | ≥4200mm | 吊顶内管线密集 |
| CSSD | ≥800mm | ≥2600mm | ≥3900mm | 灭菌器蒸汽管道占空间 |
| PIVAS | ≥1000mm | ≥2700mm | ≥4200mm | 层流天花+管线 |

> **★ 关键经验：** 手术室层如果与普通楼层共用标准层高（通常3.9-4.2m），吊顶内空间几乎必然不够。建议手术室层单独设层高≥4.5m，或设置专用设备夹层。

#### 3.1.2 洁净区围护结构

| 部位 | 要求 | 常见做法 |
|------|------|---------|
| 墙面 | 光洁、不产尘、耐擦洗、无死角 | 电解钢板/彩钢板/HPL抗菌板 |
| 吊顶 | 密封、承重（检修人员上人）、集成灯具和风口 | 电解钢板吊顶（手术室层流天花为专用产品） |
| 地面 | 无缝、防滑、防静电、耐腐蚀 | PVC卷材焊接（上翻踢脚≥100mm）/ 环氧自流平 |
| 门 | 气密、自动感应（减少接触）、观察窗 | 气密自动平移门（手术室），气密平开门（辅助用房） |
| 窗 | 气密固定窗（洁净区内不设开启扇） | 双层中空玻璃固定窗 |
| 阴阳角 | 圆弧过渡（R≥30mm） | 铝合金圆弧条 |

#### 3.1.3 人流物流气锁设计

**手术部人流路线：**
```
医护人员：更衣 → 风淋/缓冲 → 洁净走廊 → 手术室
                                    ↓
                              污物走廊 → 污物电梯（术后器械/废物）

患者：换床区（非洁净侧）→ 缓冲间 → 洁净走廊 → 手术室

★ 关键：换床区域须预留足够空间（≥3.0m×4.0m），含洁净推车存放
```

**缓冲间设置原则：**
- 洁净区与非洁净区之间：必设缓冲间
- 不同洁净等级区域之间：压差≥5Pa可不设缓冲，压差<5Pa须设缓冲
- 负压区域：必须设双道缓冲间（三区两缓）
- 缓冲间面积：≥4m²（人员通过）、≥2m²（传递窗式）
- ★缓冲间门不得同时开启：设互锁装置

### 3.2 结构专业

| 配合事项 | 要求 | 说明 |
|---------|------|------|
| 吊顶荷载 | 净化吊顶自重+检修人员荷载，约2.0-3.0kN/m² | 层流天花更重，需单独计算 |
| 设备层荷载 | 空调机组、风管等，约3.0-5.0kN/m² | 如设设备夹层须按此荷载设计 |
| 降板 | 手术室地面PVC上翻踢脚需降板50mm | 与相邻走廊齐平 |
| 管道穿楼板 | 净化区域楼板穿洞须密封处理 | 须在结构图中标注封堵要求 |
| 吊挂点 | 层流天花、大型风管吊挂点预埋 | 结构配合预埋吊杆或埋件 |

### 3.3 暖通专业

#### 3.3.1 净化空调与舒适性空调的划分

| 对比项 | 净化空调（净化公司负责） | 舒适性空调（设计院暖通专业） |
|-------|---------------------|------------------------|
| 服务区域 | 洁净区内 | 非洁净区（办公、休息、走廊等） |
| 过滤等级 | 初效+中效+高效（HEPA） | 初效+中效 |
| 风量 | 按换气次数计算，远大于舒适性 | 按冷热负荷计算 |
| 新风比 | ★手术室≥20%，负压区域100%全新风 | 按人均新风量 |
| 控制 | 温湿度+压差+风量联动控制 | 温度控制为主 |
| 划分界面 | 新风接口处（冷热源由设计院提供，新风处理后送入净化机组）| — |

> **★ 界面划分是最容易扯皮的环节：** 冷热源（冷冻水/热水）由设计院暖通专业设计并接至净化机房，净化机组及其后的风管系统由净化分包设计。新风预处理方案须双方协商确认。

#### 3.3.2 ★正负压控制策略

```
手术部压差梯度示意：

  手术室（+8Pa）→ 洁净走廊（+5Pa）→ 缓冲间（0Pa）→ 外走廊（基准0Pa）
                        ↓
                 污物走廊（-3Pa）→ 污物处理间（-5Pa）

★ 实现方式：
- 送风量 > 回风量 = 正压
- 回风量 > 送风量 = 负压
- 各房间设余压阀控制压差稳定
- ★压差传感器须在每个房间设置，并接入BMS实时监控
```

### 3.4 电气专业

| 项目 | 要求 | 说明 |
|------|------|------|
| 供电等级 | 手术室：一级负荷中的特别重要负荷 | 双路供电+UPS+柴发 |
| ★IT隔离供电 | 手术室、ICU的生命支持设备须采用IT系统 | 配绝缘监测仪，单点接地故障不跳闸 |
| 照明 | 手术室≥500lx（手术区≥20,000-100,000lx由无影灯提供）| 洁净灯具须嵌入式密封安装 |
| 应急照明 | ≥正常照度50%，持续≥180min | 手术中不可断电 |
| 静电防护 | 手术室地面电阻10⁵-10⁹Ω | 防静电PVC地面+等电位接地 |
| 接地 | 手术室设局部等电位联结端子板 | 含手术台、无影灯、监护设备 |

### 3.5 给排水专业

| 项目 | 要求 | 说明 |
|------|------|------|
| 洗手池 | 手术室区域采用感应龙头，非手触式 | 洗手池排水不得直接排入净化区地漏 |
| 地漏 | 洁净区内原则不设地漏；必须设时采用洁净地漏（带密封盖） | ★手术室内不设地漏是基本原则 |
| 纯水 | PIVAS、检验科、供应室需纯水接口 | 纯水站位置和管路由给排水专业设计 |
| 排水 | CSSD去污区排水量大（清洗机），须加大排水管径 | 建议DN100以上 |
| 蒸汽凝结水 | CSSD灭菌器凝结水排放（高温） | 须设降温池或混合冷水排放 |

## 四、施工图设计阶段

### 4.1 施工图前须确认的前置条件

| 序号 | 确认事项 | 确认方 | 时间节点 |
|------|---------|-------|---------|
| 1 | 净化分包单位确定 | 甲方/总包 | 施工图开始前 |
| 2 | 手术室等级及数量最终确认 | 院方 | 施工图开始前 |
| 3 | 净化空调冷热源方案确认 | 设计院暖通+净化分包 | 施工图开始后2周内 |
| 4 | 各净化区域吊顶标高确认（管线综合后） | 建筑+暖通+净化 | 施工图开始后4周内 |
| 5 | 净化装修材料选型确认 | 甲方+净化分包 | 方案阶段 |
| 6 | IT隔离供电方案确认 | 电气专业 | 施工图开始前 |
| 7 | 压差控制方案确认 | 暖通+净化分包 | 施工图开始后2周内 |

### 4.2 ★土建与净化分包的界面划分（关键）

| 工作内容 | 设计院（土建） | 净化分包 |
|---------|--------------|---------|
| 净化区域建筑平面布局 | ✓ 主导 | 配合优化 |
| 净化区围护结构（墙/顶/地） | 提供土建基层 | ✓ 负责净化面层 |
| 一次机电（消防、应急照明、强弱电主干） | ✓ | — |
| 净化空调系统 | 提供冷热源+新风 | ✓ 负责净化机组及末端 |
| 二次装修（洁净板、吊顶、地面） | — | ✓ |
| 二次机电（洁净灯具、医用气体面板、开关插座） | 出设计图 | 按图施工 |
| 自动控制（压差/温湿度） | 提供BA系统接口 | ✓ 负责净化区自控 |
| 医用气体 | ✓ 设计院负责 | — |
| 消防（气体灭火、喷淋等） | ✓ | — |

> **★ 经验教训：** 很多项目在界面划分上含糊不清，导致"三不管地带"出现。尤其是"洁净区内的一次消防管线由谁施工"——必须在合同中明确：**消防管线由设计院出图、施工单位安装，净化分包负责配合封堵和面层处理。**

### 4.3 常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 手术室层高不够，吊顶内挤不下 | 要么降低室内净高（不满足规范），要么改结构（代价极大） | 方案阶段锁定手术室层层高≥4.5m |
| 手术室上方有卫生间 | 漏水风险，严重违反感控要求 | ★方案阶段即检查上下层关系 |
| 净化区内设普通地漏 | 水封干涸导致细菌滋生、气流短路 | 手术室内不设地漏；必须设时用洁净密封地漏 |
| 缓冲间门未设互锁 | 两门同时开启导致压差失效 | 所有缓冲间门设电磁互锁+报警 |
| 净化空调新风比不足 | CO₂超标、麻醉废气排不出 | 手术室新风比≥20%，负压房间100%全新风 |
| CSSD流线设计可逆流 | 去污区和无菌区之间有直通门 | 严格单向流设计，三区之间仅设传递窗/互锁传递柜 |
| 生殖中心使用普通装修材料 | VOC超标导致胚胎发育异常 | ★生殖中心所有材料须通过VOC检测，且安装后须通风检测达标 |
| 净化区吊顶内管线打架 | 净高不够、检修困难 | ★施工图阶段必须做BIM管线综合 |

## 五、分包方管控

### 5.1 合同附件要求

净化分包合同须附以下技术文件作为合同附件：

1. 净化区域策划表（含各房间洁净等级、压差、换气次数）
2. 界面划分清单（明确土建/净化分包各自范围）
3. 设计院提资清单（建筑平面/吊顶标高/冷热源参数/消防点位）
4. 净化分包设计深度要求
5. 净化装修材料品牌及等级要求
6. 验收标准及检测方法

### 5.2 净化分包方案设计深度要求

| 内容 | 深度要求 |
|------|---------|
| 平面布局 | 在设计院建筑平面基础上深化净化区域内部布局 |
| 气流组织 | 每个房间的送风/回风/排风位置及风量 |
| 压差图 | 全部净化区域的压差梯度图 |
| 吊顶布置 | 灯具、风口、喷淋、维修口的综合排布 |
| 材料样板 | 墙面/吊顶/地面材料样板及色卡 |
| 空调系统图 | 净化空调原理图、风量平衡表 |
| 造价清单 | 分区域的造价明细 |

### 5.3 设计院审核净化分包图纸要点

- [ ] 各房间洁净等级是否与策划表一致
- [ ] 压差梯度是否正确（正压/负压方向）
- [ ] 层流天花尺寸是否满足规范
- [ ] 缓冲间是否设置完整（尤其负压区域）
- [ ] 送回风口位置是否合理（不能对吹，不能形成死角）
- [ ] 吊顶综合排布是否与消防点位冲突
- [ ] 净化区边界是否与建筑图一致
- [ ] 医用气体面板位置是否与净化吊顶/墙面协调
- [ ] 人流物流路线是否正确（单向流、缓冲间、互锁）

## 七、施工阶段配合

### 7.1 洁净区施工特殊要求

| 要求 | 说明 |
|------|------|
| 封闭施工 | 净化装修开始后，该区域须封闭管理，非相关人员不得进入 |

## 八、验收配合

### 8.1 检测项目及标准

| 检测项目 | 标准要求 | 检测方法 | 检测时机 |
|---------|---------|---------|---------|
| 洁净度（静态） | Ⅰ级：≥0.5μm粒子≤3520个/m³ | 粒子计数器 | 装修完成、空调运行稳定后 |
| 洁净度（动态） | 不超过静态的3倍 | 粒子计数器 | 正式运营状态下 |
| 压差 | 洁净区与非洁净区≥+5Pa，同级相邻≥+3Pa | 微压差计 | 门窗关闭状态 |
| 风速（层流） | 截面平均风速0.25-0.35m/s | 风速仪 | — |
| 换气次数 | 见策划表各区域要求 | 风速仪+截面积计算 | — |
| 温度 | 22-25°C（手术室）、24±1.5°C（ICU） | 温度计 | 空调运行稳定1h后 |
| 湿度 | 40-60%RH（手术室） | 湿度计 | 同上 |
| 噪声 | 手术室≤52dB(A)、ICU≤45dB(A) | 声级计 | 空调运行状态 |
| 照度 | 手术室≥500lx（术野另计） | 照度计 | 全部灯具开启 |
| 细菌浓度 | Ⅰ级手术区≤5CFU/30min·Φ90皿 | 沉降法/浮游法 | 动态检测 |

### 8.2 验收阶段设计院须配合的工作

| 工作内容 | 负责专业 |
|---------|---------|
| 确认净化区域平面与设计图一致 | 建筑 |
| 确认结构吊挂安全 | 结构 |
| 参与风量/压差调试 | 暖通 |
| 确认IT系统绝缘监测正常 | 电气 |
| 确认给排水接口无渗漏 | 给排水 |
| 提供设计文件作为验收依据 | 全专业 |

## 九、全过程检查清单（Checklist）

**方案阶段：**
- [ ] 净化区域策划表已编制并经院方确认
