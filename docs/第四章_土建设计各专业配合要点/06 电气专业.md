### 6.1 强电

| 要点 | 要求 |
|------|------|
| ★医院供电等级 | 一级负荷（手术室/ICU/急诊/产房/血透/NICU为一级中的特别重要负荷） |
| 双路供电 | 两路10kV独立电源+柴油发电机 |
| UPS | 手术室/ICU/血透/信息机房配UPS |
| ★IT隔离供电 | 手术室/ICU生命支持设备采用IT系统（单点接地故障不跳闸） |
| 变配电室 | 宜设在负荷中心附近，减少线路损耗 |

### 6.2 弱电

| 要点 | 要求 |
|------|------|
| 弱电系统 | 详见5.19弱电智能化专项 |
| 综合布线 | ★医院信息点密度是普通办公的2-3倍 |
| 弱电井 | 每层每分区≥6m²，上下对齐 |

### 6.3 室外总体（电气）

| 要点 | 要求 |
|------|------|
| 强电（市政电） | 10kV双路引入，变电站选址 |
| 弱电（电信网络） | 光纤引入、通信管道预留 |

### 6.4 错题本

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 手术室未设IT系统 | 接地故障跳闸导致手术中断 | ★IT系统为手术室必备 |
| 柴发容量不足 | 停电时无法满足全部一级负荷 | 按实际一级负荷计算+30%余量 |
| 弱电井太小 | 电缆拥挤、散热差 | 医院弱电井≥常规×1.5倍 |
