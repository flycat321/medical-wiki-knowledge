# 5.06 磁屏蔽与防辐射设计管理细则

---

## 一、概述

医疗建筑中涉及辐射防护的科室和设备种类繁多，是EPC项目中**技术复杂度最高、多方配合最密集**的专项之一。

### 涉及的科室与设备

| 科室 | 设备 | 辐射/屏蔽类型 |
|------|------|-------------|
| 放射科 | DR、CT（普通/高端/双源）、DSA | X射线防护 |
| 核医学科 | PET-CT、PET-MR、SPECT、回旋加速器 | γ射线+正电子防护 |
| 放疗科 | 直线加速器、伽马刀、后装治疗机、质子/重离子（大型） | 高能X射线+中子防护 |
| 影像中心 | MRI（1.5T/3.0T/7.0T） | ★磁屏蔽（非辐射） |
| 介入中心 | DSA（心血管/神经/外周） | X射线防护 |
| 口腔科 | CBCT、全景机 | X射线防护（轻量级） |
| 超声科 | 无辐射 | 不涉及 |

### 法规前置条件

| 事项 | 主管部门 | 时间节点 | 说明 |
|------|---------|---------|------|
| 辐射安全许可证 | 生态环境部门 | 项目建设前申请 | 使用放射源和射线装置须持证 |
| 环境影响评价 | 生态环境部门 | 开工前取得批复 | 含辐射环评专篇 |
| 辐射防护预评价 | 有资质单位编制 | 初设阶段 | 计算各机房防护厚度的依据 |
| 辐射防护竣工验收 | 生态环境部门 | 设备安装后、投用前 | 现场实测辐射剂量 |

**核心原则：防护设计必须先行——在土建施工图出图前，防护方案须确定，否则结构降板、墙体加厚等均无法落实。**

### 2.2 需院方确认的事项

- [ ] 各类影像/放疗设备的品牌和型号（★越早确定越好，不同品牌机房条件差异巨大）

## 三、初步设计阶段

### 3.1 建筑专业

#### 3.1.1 防护区平面布局原则

```
辐射防护分区原则：

  控制区（机房内）──→ 监督区（操作间、走廊）──→ 非限制区（公共区域）

  ★关键：
  1. 控制区边界处的剂量必须满足限值
  2. 监督区边界处的剂量应≤限值的1/4
  3. 机房门口设辐射警示灯和声光报警
  4. 机房相邻房间（上下左右）均须验算防护是否达标
```

#### 3.1.2 ★防护区域上下层关系处理

| 问题 | 处理方式 |
|------|---------|
| 防护区上方楼板 | 结构降板，预留硫酸钡砂浆厚度（通常50-100mm）|
| 防护区下方楼板 | 视设备类型，CT等一般不需要；直线加速器须加厚底板 |
| 防护区上方为可利用空间 | 楼板须按防护要求加铅板或加厚混凝土 |
| 防护区上方为屋面/设备层 | 减少防护要求，但仍需验算 |

> **★ 关键经验：** 降板50mm对普通CT够用，但对高端CT可能不够（需60-80mm硫酸钡）。必须在防护预评价报告出来后再确定降板深度，不能凭经验拍。

#### 3.1.3 ★CT等大型设备运输路径规划

```
运输路径验证checklist：

1. 从建筑物外部入口到机房的完整路径标注
2. 路径上所有门洞净宽≥设备包装宽度+200mm
3. 路径上所有转弯处验证转弯半径
4. 路径上楼板荷载≥设备运输重量（含运输工具）
5. 是否有台阶？（不允许有台阶，坡道坡度≤1:8）
6. 是否需要预留吊装口？（MRI磁体通常需要）
7. 路径上不得有不可拆除的障碍物

★ 建议：在施工图中单独出一张"大型设备运输路径图"
```

#### 3.1.4 迷道设计（直线加速器/核医学）

迷道是利用辐射在空气中的衰减来替代部分屏蔽材料的通道设计：

| 参数 | 直线加速器 | 核医学给药区 |
|------|----------|------------|
| 迷道形式 | L型或Z型 | L型 |
| 迷道宽度 | ≥1.8m（需通过治疗床） | ≥1.2m |
| 迷道长度 | 6-12m（视加速器能量） | 3-5m |
| 迷道壁厚 | ≥1.0m混凝土 | ≥300mm混凝土+铅 |
| 迷道末端 | 设防护门（通常≥8吨重） | 设防护门 |
| ★注意 | 迷道入口方向不得正对治疗束 | — |

### 3.2 结构专业

| 配合事项 | 要求 |
|---------|------|
| 防护区降板 | 各机房根据防护预评价报告确定降板深度，不得统一按50mm |
| CT/DSA机房荷载 | 局部荷载按设备基础图计算，一般≥8-10kN/m² |
| MRI机房荷载 | ★局部≥20-25kN/m²（磁体），且须验算楼板振动（VC级） |
| 直线加速器结构 | 1.5-2.5m厚混凝土墙和顶板，属于特种结构设计 |
| MRI结构 | ★5高斯线范围内结构钢筋须采用非磁性材料（不锈钢筋或玻纤筋）|
| 设备基础 | CT/MRI均需独立设备基础，与主体结构脱开（减振） |
| 防护门基础 | 直线加速器防护门重达8-15吨，门框和地轨须专门设计基础 |

### 3.3 暖通专业

| 区域 | 要求 |
|------|------|
| CT/DSA机房 | 精密空调（温度22-24°C、湿度40-60%），设备散热量大 |
| MRI机房 | ★失超排气管：直径≥250mm，直通室外，出口远离人员活动区 |
| MRI设备间 | 冷水机散热量极大，须单独设排风或水冷 |
| 直线加速器 | 治疗室通风（微正压防止放射性气体外泄）；迷道通风 |
| 核医学 | ★全区域负压通风，排风经活性炭过滤后高空排放 |
| 核医学给药室 | 负压，独立排风系统，排风口设在建筑最高点 |
| PET-CT候诊间 | 每间独立排风（患者体内有放射性药物） |

### 3.4 电气专业

| 区域 | 要求 |
|------|------|
| CT | 专用回路，80-200kVA，短时大电流（扫描瞬间） |
| MRI | ★专用回路+滤波器（EMI滤波），防止外部电磁干扰 |
| MRI屏蔽室 | 所有进出屏蔽室的电缆须经穿墙滤波器（波导管） |
| 直线加速器 | 专用变压器，200-500kVA，电压稳定性≤±2% |
| 辐射警示灯 | 所有放射设备机房门口设辐射警示灯，与设备开机联锁 |
| 门禁联锁 | 设备运行时机房门锁定，强行开门则设备停机 |
| 监控 | 治疗室/操作间CCTV监控，操作人员须能看到患者 |

### 3.5 给排水专业

| 项目 | 要求 |
|------|------|
| ★衰变池 | 核医学科必设，容积按日排放量×10倍衰变周期计算 |
| 衰变池位置 | 地下室或室外地下，远离人员活动区 |
| 放射性废水管道 | 专用管道，不得与普通排水合流，材质耐腐蚀（HDPE）|
| 核医学卫生间 | 患者专用卫生间，排水接入衰变池 |
| MRI冷却水 | 专用冷却水管路，水质要求高 |

## 四、施工图设计阶段

### 4.1 施工图前须确认的前置条件

| 序号 | 确认事项 | 确认方 | 时间节点 |
|------|---------|-------|---------|
| 1 | 设备品牌型号确定 | 院方 | ★越早越好，最迟初设阶段 |
| 2 | 辐射防护预评价报告 | 有资质单位 | 施工图开始前 |
| 3 | 防护设计方案（含各面防护厚度） | 防护公司 | 施工图开始前 |
| 4 | MRI屏蔽方案 | 屏蔽公司 | 施工图开始前 |
| 5 | 设备基础图（厂家提供） | 设备厂家 | 施工图开始后2周内 |
| 6 | 运输路径确认 | 建筑+设备厂家 | 施工图阶段 |

| 工作内容 | 设计院 | 防护公司 | 说明 |
|---------|-------|---------|------|
| 机房建筑平面布局 | ✓ 主导 | 配合 | — |

> **★ 原文强调："特装区阴影里的所有影像检查房间如CT、核磁等二次设计指的防辐射公司只做辐射防护设计，土建和精装还是咱们这边做，气体也是我们要做的。"**

### 4.3 常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 设备品牌未定就出施工图 | 机房尺寸、防护厚度、荷载全部可能需要改 | ★坚持设备型号确定后再出防护区施工图 |
| 防护区上方降板不够 | 硫酸钡砂浆挤占室内净高，或防护不达标 | 按预评价报告确定降板深度 |
| MRI 5高斯线范围内有普通钢筋 | 磁场畸变、影像质量下降 | 5高斯线范围内结构筋改非磁性材料 |
| CT运输路径上有台阶 | 设备无法就位 | ★出运输路径图，逐段验证 |
| MRI失超排气管未设或管径不够 | 液氦泄漏导致窒息风险（致命） | ★必须设置，管径≥250mm，直通室外 |
| 核医学排水未接衰变池 | 放射性废水直排市政管网，违法 | 核医学区所有排水（含卫生间）接衰变池 |
| 直线加速器迷道方向正对治疗束 | 迷道口辐射剂量超标 | 迷道入口方向与治疗束方向成90°或更大角度 |
| 防护门门框未预埋 | 后期凿洞安装影响防护完整性 | 防护门框随结构施工一起预埋 |

## 五、分包方管控

### 5.1 防护公司资质要求

- 持有辐射防护相关资质（甲/乙级）
- 有同等级医疗项目业绩
- 具备防护预评价和控制效果评价资质（或与有资质单位合作）

### 5.2 防护公司须提交的设计成果

| 文件 | 内容 | 审核要点 |
|------|------|---------|
| 防护预评价报告 | 各机房各面（四墙+顶+地）的防护当量 | 是否按最终设备参数计算 |
| 防护施工图 | 铅板/硫酸钡范围、厚度标注 | 是否覆盖完整，搭接方式 |
| 铅门/铅窗选型 | 型号、防护当量、尺寸 | 是否与建筑门洞匹配 |
| 屏蔽室方案（MRI） | 屏蔽材料、层数、接口处理 | 是否满足设备厂家屏蔽要求 |
| 施工方案 | 铅板安装工艺、硫酸钡配比及施工工艺 | — |

## 六、厂家配合（设备厂家 + 防护公司 双线配合）

```
双线配合时序：

  设备厂家 ──→ 提供机房条件书 ──→ 建筑方案调整 ──┐
                                                    ├──→ 施工图出图
  防护公司 ──→ 防护预评价报告 ──→ 防护方案确定 ──┘

  ★ 关键时间节点：
  1. 设备型号确定 → 厂家出机房条件书（2周内）
  2. 机房条件书 → 防护公司出预评价报告（4周内）
  3. 预评价报告 → 设计院调整施工图（2周内）
  4. 施工图确认 → 防护公司出防护施工图（2周内）

  ★ 如果设备型号迟迟不定，按该类型设备最大参数预留防护条件
```

## 七、施工阶段配合

### 7.1 防护施工特殊要求

| 工序 | 要求 | 质量控制要点 |
|------|------|------------|
| 铅板安装 | 满焊/搭接≥30mm，不得有针孔和裂缝 | ★搭接处是薄弱环节，须重点检查 |
| 硫酸钡砂浆 | 配比严格（比重≥3.2g/cm³），分层施工每层≤20mm | ★抗裂是重点：加钢丝网，养护≥14天 |
| 铅门安装 | 门框与墙体铅板搭接完整，铰链承重足够 | 直线加速器防护门需电动/气动，须调试 |
| MRI屏蔽室 | 铜板/钢板焊接密封，所有接缝连续焊接 | ★屏蔽效能测试须在封闭后进行 |
| 穿墙管线 | 所有穿过防护墙的管线须加铅套管并密封 | 遗漏一处即防护失效 |
| 隐蔽工程 | 铅板安装完成、封面层之前须验收 | ★必须留影像资料（照片/视频） |

### 7.2 各专业施工配合

| 节点 | 建筑 | 结构 | 暖通 | 电气 | 给排水 |
|------|------|------|------|------|--------|
| 主体施工 | 降板验收 | 设备基础预留 | — | — | 衰变池施工 |
| 防护施工前 | 验收基层 | 预埋件验收 | 穿墙套管预埋 | 穿墙套管预埋 | 穿墙套管预埋 |
| 铅板安装 | 配合墙面基层 | — | — | — | — |
| 隐蔽验收 | ★必须参与 | — | 确认套管密封 | 确认套管密封 | 确认套管密封 |
| 面层施工 | 精装面层 | — | — | — | — |
| 设备就位 | 运输路径清障 | — | 空调开启 | 送电调试 | 冷却水接通 |

## 八、验收配合

### 8.1 辐射防护检测

| 检测项目 | 检测方法 | 合格标准 | 检测时机 |
|---------|---------|---------|---------|
| 屏蔽体表面剂量率 | 辐射剂量仪 | 控制区边界≤2.5μSv/h | 设备安装后、正式使用前 |
| 人员年有效剂量 | 计算+实测 | 公众≤1mSv/年，职业≤20mSv/年 | 同上 |
| MRI磁场泄漏 | 高斯计 | 5高斯线不超出屏蔽室+标注范围 | 磁体励磁后 |
| MRI射频屏蔽效能 | 频谱分析仪 | ≥80dB（视设备要求） | 屏蔽室封闭后 |
| 辐射警示联锁 | 功能测试 | 设备开机→警示灯亮→门锁联锁 | 设备安装后 |

### 8.2 环评验收

- 辐射防护竣工验收由生态环境部门组织
- 须提交：防护预评价报告、防护施工图、第三方检测报告、设备清单
- ★验收通过后方可正式投入使用
