# 5.11 电梯专项设计管理细则

---

## 一、概述

电梯是医疗建筑中使用频率最高、对运营效率影响最大的垂直交通系统。医院电梯不同于普通公建，存在以下特殊性：

- **类型多**：客梯、病床梯、手术专用梯、污物梯、餐梯/传菜梯、消防梯、货梯、担架梯，一个三甲医院可能同时涉及6-8种类型
- **流线要求严**：洁污分流、医患分流，部分电梯需前后贯通（前门进后门出）
- **运行时间长**：24小时不间断运行，对可靠性要求极高
- **荷载大**：病床梯需容纳病床+护理人员+设备，轿厢尺寸和载重远超普通客梯
- **与多个专项设计交叉**：物流系统、消防、智能化、精装修均有接口

**核心原则：电梯选型和井道条件必须在方案阶段锁定，施工图阶段不应再调整井道尺寸。**

---

## 二、策划阶段（可研/方案设计）

### 2.1 项目负责人/建筑专业须完成的工作

#### 2.1.1 电梯配置方案策划

需在方案阶段与院方确认以下内容，形成《电梯配置策划表》：

| 确认事项 | 说明 | 常见问题 |
|---------|------|---------|
| 电梯总数量 | 按日门诊量、床位数、楼层数综合计算 | 院方后期加梯导致结构大改 |
| 各类型数量分配 | 客梯、病床梯、污物梯、货梯、消防梯等 | 病床梯数量不足是最常见的运营投诉 |
| 电梯分区 | 门诊区、住院区、医技区、后勤区分组 | 不分区会导致住院部电梯被门诊人流占满 |
| 前后贯通需求 | 哪些电梯需要对穿门（前进后出） | 方案阶段遗漏，施工图阶段改井道极其被动 |
| 无障碍电梯 | 至少每组设1台，符合GB 50763要求 | 轿厢深度不够轮椅调头 |
| 消防电梯 | 数量、位置须满足GB 50016要求 | 与消防分区划分联动 |

#### 2.1.2 电梯数量估算参考

**住院楼（按床位数估算）：**
- 每150-200床设1台病床梯（载重≥1600kg，建议2000kg）
- 每200-250床设1台污物/餐梯
- 消防电梯按防火分区规范配置

**门诊楼（按日门诊量估算）：**
- 日门诊量3000人次以下：客梯4-6台
- 日门诊量3000-5000人次：客梯6-8台
- 每栋建筑至少1台担架梯（非病床梯也需满足担架进入）

**医技楼：**
- 至少1台大吨位货梯（用于大型设备运输，载重≥3000kg，建议5000kg）
- 注意：CT、MRI等设备运输路径可能不经过常规电梯，需单独验证

#### 2.1.3 竖向交通流线策划

```
需在方案阶段明确的流线分离原则：

┌──────────────────────────────────────────────────┐
│  医院竖向交通"四分离"原则                          │
│                                                    │
│  1. 患者/家属 与 医护人员 分离                      │
│  2. 洁物（药品/无菌物品）与 污物（垃圾/被服）分离    │
│  3. 门诊人流 与 住院人流 分离                       │
│  4. 活人通道 与 遗体通道 严格分离                    │
│                                                    │
│  太平间专用梯：必须有独立路径，不得与患者动线交叉     │
└──────────────────────────────────────────────────┘
```

#### 2.1.4 需院方书面确认的事项

- [ ] 电梯品牌偏好或限制（进口/合资/国产，部分医院有集团采购指定品牌）
- [ ] 是否需要担架梯兼做消防梯
- [ ] 手术室专用梯的使用管理模式（刷卡控制/专人管理）
- [ ] 太平间电梯是否需要独立出入口
- [ ] 是否需要配置智能群控/目的层调度系统
- [ ] 餐梯/传菜梯是否与箱式物流系统合并

### 2.2 结构专业须提前介入的工作

- 电梯井道位置对结构体系的影响评估（剪力墙布置、核心筒设计）
- 机房层/无机房方案对屋面结构的影响
- 大吨位货梯的基坑深度和底坑荷载预估（5000kg货梯底坑深度≥1800mm）
- 消防电梯前室结构布局

### 2.3 限额设计阶段的造价预控

| 电梯类型 | 单台造价参考（含安装） | 说明 |
|---------|---------------------|------|
| 乘客梯（1000kg/1.75m/s） | 25-40万 | 品牌差异大 |
| 病床梯（1600kg/1.0m/s） | 35-55万 | 轿厢加深，门宽≥1400mm |
| 病床梯（2000kg/1.0m/s，前后贯通） | 45-70万 | 对穿门，井道尺寸显著加大 |
| 货梯（3000-5000kg） | 50-80万 | 含加强型轿厢底板 |
| 餐梯/传菜梯（300-500kg） | 8-15万 | 杂物梯标准 |
| 消防梯（1000-1600kg） | 30-50万 | 含前室加压送风配合 |
| 担架梯 | 30-45万 | 轿厢深度≥2100mm |

> **注意：** 以上为设备+安装费用，不含井道土建、电气配电及装饰费用。概算编制时需单列"电梯井道土建"和"电梯轿厢装饰"两项。

## 三、初步设计阶段

### 3.1 建筑专业

#### 3.1.1 井道平面尺寸确定

**关键原则：井道净尺寸必须满足"最大包络"——即满足主流品牌（奥的斯、通力、三菱、日立、迅达）中对井道要求最大的那个，确保招标时不因井道限制而排除品牌。**

各类型电梯井道参考尺寸（净尺寸，mm）：

| 类型 | 载重(kg) | 轿厢宽×深 | 井道宽×深（参考） | 开门宽度 | 备注 |
|------|---------|-----------|-----------------|---------|------|
| 客梯 | 1000 | 1600×1500 | 2200×2200 | 900 | — |
| 客梯 | 1350 | 1800×1500 | 2400×2200 | 1000 | — |
| 病床梯 | 1600 | 1500×2400 | 2200×3000 | 1400 | 标准病床梯 |
| 病床梯（贯通） | 2000 | 1600×2600 | 2400×3200 | 1400（双侧） | 前后双开门 |
| 货梯 | 3000 | 2000×3000 | 2800×3600 | 1500 | — |
| 货梯 | 5000 | 2400×3400 | 3200×4000 | 1800 | 大型设备运输 |
| 担架梯 | 1000 | 1100×2100 | 1800×2800 | 900 | GB 50763要求 |

> **★ 重要提示：** 以上为经验参考值，不同品牌、不同速度、不同开门方式（中分/旁开）差异可达200-400mm。初设阶段建议按上述尺寸+200mm余量预留，待厂家确定后精确调整。

#### 3.1.2 井道剖面尺寸确定

| 参数 | 要求 | 常见问题 |
|------|------|---------|
| 顶层高度（OH值） | 客梯≥4200，病床梯≥4500，货梯≥4800（速度≤1.75m/s时） | 顶层层高不够是最常见的"硬伤"，改结构代价极大 |
| 底坑深度（PD值） | 客梯≥1500，病床梯≥1600，货梯≥1800（有缓冲器空间） | 底坑与地下室底板标高冲突，需结构配合加深 |
| 层高变化 | 每层开门位置标高须标注在井道图上 | 医院常出现首层层高>标准层，漏标导致导轨加工错误 |
| 井道壁厚 | 混凝土井道≥200mm（一般取250mm） | 砌体井道在医院不推荐（设备振动+防火要求） |

#### 3.1.3 电梯厅设计要点

| 位置 | 设计要点 |
|------|---------|
| 门诊电梯厅 | 厅深≥轿厢深度+1500mm（大量候梯人流），设无障碍按钮高度≤1100mm |
| 住院部电梯厅 | 厅深≥轿厢深度+2000mm（病床推出后需调头），地面防滑 |
| 手术层电梯厅 | 洁净区/非洁净区界面处理，刷卡门禁，换床区域预留 |
| 地下室电梯厅 | 排水沟设置（消防电梯前室），防火卷帘/防火门位置不得遮挡梯口 |
| 太平间电梯厅 | 独立通道，视线遮挡，不应开向公共区域 |
| 货梯厅 | 厅前净宽≥3000mm（叉车转弯），地面耐磨，设防撞角 |

#### 3.1.4 初设阶段必须标注在图纸上的内容

- 井道平面尺寸（净尺寸）
- 各层开门方向及开门宽度
- 前后贯通梯的标注
- 顶层OH值及底坑PD值
- 机房位置（有机房）或控制柜位置（无机房）
- 消防电梯前室面积标注

### 3.2 结构专业

- 井道壁为结构墙（剪力墙或框架+填充墙），不建议后期开洞
- 电梯基坑底板厚度及配筋（考虑缓冲器冲击荷载）
- **底坑抗浮验算**（地下水位高的地区，空井道底坑可能上浮）
- 机房楼板荷载（有机房方案：主机+控制柜+曳引轮，局部荷载可达10kN/m²）
- 预埋件位置（导轨支架、层门门框）——初设阶段标注原则，施工图阶段由厂家出深化图后精确定位

### 3.3 电气专业

- 各电梯供电回路的回路编号和配电容量预估
- 消防电梯的双路供电+末端自动切换要求
- 电梯机房/控制柜的空调或通风预留（控制柜工作温度5-40°C）
- 五方通话系统的弱电预留
- 电梯监控（轿厢内摄像头）的弱电预留
- **电梯能量回馈装置的配电方案**（如有节能要求）

### 3.4 暖通专业

- 电梯机房通风或空调（机房温度≤40°C，否则主机保护停机）
- 消防电梯前室的加压送风设计
- 电梯井道作为竖向风道的烟囱效应处理（超高层尤其注意）

### 3.5 给排水专业

- 消防电梯底坑集水坑+排水泵（排水量≥10L/s，GB 50016要求）
- 底坑防水做法（尤其地下水位高的地区）
- 电梯机房是否需要设给排水点位（一般不需要，但确认有无屋面排水管经过机房上方）

## 四、施工图设计阶段

### 4.1 建筑专业

#### 4.1.1 施工图阶段须确认的前置条件

| 序号 | 确认事项 | 确认方 | 时间节点 |
|------|---------|-------|---------|
| 1 | 电梯品牌/型号确定 | 甲方/总包 | **施工图开始前** |
| 2 | 各台电梯的精确技术参数表（厂家提供） | 电梯厂家 | 施工图开始后2周内 |
| 3 | 厂家井道布置图（平面+剖面） | 电梯厂家 | 施工图开始后2周内 |
| 4 | 预埋件图纸 | 电梯厂家 | 结构施工图出图前 |
| 5 | 电气参数（功率、电流、电压、接地要求） | 电梯厂家 | 电气施工图出图前 |
| 6 | 轿厢装饰方案 | 精装分包/甲方 | 精装施工图出图前 |
| 7 | 智能群控/目的层调度方案 | 电梯厂家+弱电 | 弱电施工图出图前 |

> **★ 经验教训：** 如果甲方/总包迟迟不确定电梯品牌，项目负责人须书面发函催促，并明确告知"逾期未定则按XX品牌参数出图，后期因品牌变更导致的井道修改费用另计"。同时施工图应按"最大包络"原则预留，在图中注明"井道净尺寸待厂家确认后以深化图为准"。

#### 4.1.2 施工图图纸内容

**电梯间详图须包含：**

1. **井道平面详图**（1:50）
   - 井道净尺寸（标注到结构面）
   - 开门方向、门洞宽度×高度
   - 轿厢投影位置（虚线）
   - 对重位置
   - 导轨位置（厂家图纸转绘）
   - 井道壁上预留孔洞（强弱电穿管、通风洞口）

2. **井道剖面图**（1:50或1:100）
   - 各层开门标高
   - 顶层净高OH值标注
   - 底坑深度PD值标注
   - 机房层净高（有机房时≥2800mm）
   - 吊钩位置及荷载标注（检修用，一般要求承载≥主机重量的2倍）
   - 底坑集水坑位置

3. **电梯厅详图**
   - 层门门套做法
   - 电梯厅地面做法（含防滑等级）
   - 呼梯盒/楼层显示器预留位置及尺寸
   - 无障碍按钮位置标高

4. **机房平面图**（有机房方案）
   - 主机位置、控制柜位置
   - 通风/空调位置
   - 配电箱位置
   - 检修空间（主机周围≥800mm通道）
   - 承重梁位置（需结构配合）

#### 4.1.3 常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| OH值不够 | 电梯无法安装，需抬高屋面或降低顶层地面 | 方案阶段即核算，建筑与结构共同确认 |
| 底坑深度不够 | 需局部加深基础，影响结构安全及防水 | 初设阶段锁定各梯型底坑深度 |
| 病床梯井道深度不足 | 2000kg病床梯改为1600kg，运营能力下降 | 按前后贯通2000kg预留，后期可降级但不可升级 |
| 层门口装修面层挤占开门宽度 | 病床推不进电梯 | 门洞宽度标注须为"装饰完成面净宽" |
| 消防电梯前室面积不达标 | 消防验收不通过 | 合用前室≥12m²，独立前室≥6m² |
| 电梯厅深度不足 | 病床出梯无法转弯 | 住院部电梯厅深≥轿厢深+2000mm |
| 未标注吊钩位置 | 施工单位自行焊接，承载力不够 | 剖面图中明确标注，结构配合预埋 |

### 4.2 结构专业

#### 4.2.1 施工图须完成的内容

- 井道壁配筋图（剪力墙/配筋砌体）
- 底坑底板配筋及防水做法
- **导轨支架预埋件**：须根据厂家提供的预埋件图精确定位，每层2-4组，位置偏差≤10mm
- **门框预埋件**：每层开门位置的门框预埋角钢
- 机房承重梁（有机房方案，承载主机+曳引轮动荷载）
- 机房吊钩预埋（荷载标注于图纸）
- 井道内照明预埋管线

#### 4.2.2 关键配合节点

```
结构出图时序要求：

  电梯厂家提供深化图 ──→ 建筑校核井道尺寸 ──→ 结构绘制预埋件 ──→ 结构出图
       ↑                                              ↓
  （厂家确定后2周内）                          （须在该层结构施工前完成）

  ★ 如果结构已出图但厂家未定，须在图中注明：
    "电梯导轨支架及门框预埋件待厂家确定后出补充图，施工时预留后埋条件"
```

#### 4.2.3 常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 预埋件遗漏或位置偏差>20mm | 后期植筋、打膨胀螺栓，影响结构安全及观感 | 必须拿到厂家深化图后再出预埋图 |
| 底坑底板未考虑缓冲器冲击荷载 | 底板开裂 | 按厂家提供的缓冲器反力值计算 |
| 井道壁预留洞未封堵 | 影响井道气密性，高速梯啸叫 | 穿管完成后用防火材料封堵 |
| 机房承重梁跨度/截面不够 | 主机振动传递，楼板共振 | 按厂家提供的主机基础图校核 |

### 4.3 电气专业

#### 4.3.1 强电设计要点

| 项目 | 要求 | 说明 |
|------|------|------|
| 供电等级 | 消防梯：一级负荷；病床梯/客梯：二级负荷 | 医院所有电梯建议按不低于二级负荷供电 |
| 供电电压 | 380V/三相五线制（常规），部分进口梯需确认 | 厂家参数表确认 |
| 单台功率 | 客梯15-30kW，病床梯20-40kW，货梯40-80kW | 按厂家参数确认，变频梯启动电流约为额定的1.5-2倍 |
| 配电位置 | 机房内（有机房）/ 井道顶部控制柜旁（无机房） | 电缆从配电室到机房/控制柜的路由须提前确认 |
| 接地 | PE线直接引至机房/控制柜，接地电阻≤4Ω | 含轿厢、门框、导轨的等电位联结 |
| 照明 | 井道内每隔7m设1个灯位（底坑和顶部各1个），机房按100lx | GB 7588要求 |
| 插座 | 机房/控制柜旁至少1个三孔插座 | 检修用 |

#### 4.3.2 弱电设计要点

| 系统 | 要求 |
|------|------|
| 五方通话 | 轿厢↔机房↔管理室↔底坑↔轿顶，五方均设通话终端 |
| 轿厢监控 | 每台电梯轿厢内1路摄像头，井道内预留视频线至弱电井 |
| 电梯运行监控 | 接入BAS/BA系统，实时显示各梯运行状态、故障报警 |
| 智能群控 | 如采用目的层调度，各层设目的层选层器，弱电须配合布线 |
| 门禁系统 | 手术室层、太平间层、药库层等限制楼层的刷卡控制 |
| 紧急广播 | 消防电梯轿厢内设消防广播扬声器 |
| 医护呼叫联动 | 部分项目要求病床梯与护理呼叫系统联动（可选） |

#### 4.3.3 常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 配电容量按额定功率配，未考虑启动电流 | 频繁跳闸 | 按厂家提供的最大运行电流×1.25选断路器 |
| 消防梯未设双路末端自动切换 | 消防验收不通过 | 在配电箱内设ATS |
| 弱电预留线管未到位 | 后期明装线槽，影响观感 | 每台梯从轿顶到弱电井至少预留2根DN25管 |
| 无机房电梯控制柜位置未预留电源 | 后期拉明线 | 无机房方案须在顶层井道旁设专用配电箱 |

### 4.4 暖通专业

#### 4.4.1 须完成的设计内容

| 位置 | 设计要求 | 说明 |
|------|---------|------|
| 机房通风 | 排风量按散热量计算，一般≥6次/h换气 | 机房温度≤40°C，否则主机热保护停机 |
| 机房空调 | 寒冷/严寒地区或内区机房须设空调，温度5-40°C | 冬季也需保温，低温影响润滑油粘度 |
| 消防前室加压送风 | 按GB 51251设计，风量计算须考虑门开启面积 | 注意：合用前室与独立前室的风量标准不同 |
| 井道通风 | 超高层井道顶部宜设通风口，面积≥井道截面1% | 减少活塞效应 |
| 风管路由 | **风管不得穿越电梯井道** | 强制条文，建筑复核 |

### 4.5 给排水专业

| 项目 | 要求 | 说明 |
|------|------|------|
| 底坑排水 | 消防电梯底坑设集水坑（≥0.5m³），配排水泵（≥10L/s） | GB 50016强制要求 |
| 底坑防水 | 防水等级不低于地下室其他部位 | 渗漏会腐蚀导轨和缓冲器 |
| 管道避让 | **给排水管道严禁穿越电梯井道和机房** | 强制条文 |
| 屋面排水 | 机房上方屋面排水不得从机房内穿过 | 漏水导致电气故障 |
| 消防前室 | 合用前室内消火栓位置不得影响电梯门开启 | 现场布置时常被忽略 |

└─────────────────────────────────────────────────────────────────┘

### 5.2 设计院审核厂家图纸的要点

**建筑专业审核：**
- 井道尺寸是否与施工图一致（允许偏差：井道净尺寸≥厂家要求值）
- 开门方向、开门宽度是否正确
- 各层标高是否与建筑标高一致
- 前后贯通门的位置是否正确
- 呼梯盒/显示器位置是否与精装方案冲突

**结构专业审核：**
- 预埋件数量、位置、规格是否合理
- 机房承重梁荷载是否在结构承载范围内
- 吊钩承载力是否满足
- 是否有后开洞需求（须加固处理）

**电气专业审核：**
- 电气功率、电压是否与配电方案一致
- 接地方式是否符合设计方案
- 消防联动接口是否明确
- 弱电接口是否完整

### 5.3 厂家配合常见问题

| 问题 | 应对措施 |
|------|---------|
| 厂家迟迟不提供深化图 | 项目经理书面催促，抄送甲方/总包，设定deadline |
| 厂家深化图与设计院施工图有冲突 | 组织三方会审，以设计院施工图为基准，厂家调整 |
| 甲方中途换品牌 | 建筑专业重新校核井道尺寸，如需改井道须评估影响并发变更 |
| 厂家提出井道加大需求 | 评估对相邻房间的影响，不能简单同意，须建筑/结构联合评估 |
| 进口梯与国标有差异 | 以国标为底线，厂家标准高于国标的从厂家，低于国标的须整改 |

## 六、施工阶段配合

### 6.1 各专业须配合的工作

#### 6.1.1 建筑专业

| 施工节点 | 配合内容 |
|---------|---------|
| 基础施工 | 确认底坑深度、集水坑位置，**实测净尺寸并书面记录** |
| 主体施工 | 每层预埋件位置旁站或抽检，门洞预留尺寸检查 |
| 二次结构 | 井道壁抹灰垂直度检查（偏差影响导轨安装） |
| 装修阶段 | 层门门套安装配合，电梯厅地面标高与轿厢地面齐平 |
| 安装完成 | 轿厢内装饰验收 |

#### 6.1.2 结构专业

| 施工节点 | 配合内容 |
|---------|---------|
| 底坑施工 | 底板厚度、配筋、防水做法确认 |
| 主体施工 | 预埋件定位复核（偏差>15mm须提出整改方案） |
| 机房层 | 承重梁/吊钩安装验收 |
| 后开洞 | 如有后开洞需求，出具加固方案并监督执行 |

#### 6.1.3 电气专业

| 施工节点 | 配合内容 |
|---------|---------|
| 主体施工 | 预埋管线到位确认（强电到机房/控制柜，弱电到各层） |
| 安装前 | 电梯配电箱送电测试 |
| 安装中 | 临时用电协调 |
| 安装后 | 消防联动调试配合（迫降、返回首层、非消防电源切断） |
| 验收前 | 接地电阻测试 |

### 6.2 施工阶段常见问题与处置

| 问题 | 发生阶段 | 处置 |
|------|---------|------|
| 底坑渗水 | 基础/地下室施工 | 设计院出底坑加强防水方案，施工单位执行 |
| 预埋件偏差过大 | 主体施工 | 偏差≤30mm：植化学锚栓补救；偏差>30mm：结构出加固方案 |
| 井道垂直度偏差>25mm | 主体完成 | 影响导轨安装，需凿除修补或加厚找平层 |
| 层门口标高偏差 | 装修阶段 | 调整门槛高度（允许调整范围±20mm），超范围须改地面标高 |
| 机房温度过高 | 安装调试阶段 | 暖通专业核算通风量，增设排风扇或空调 |
| 消防联动失败 | 调试阶段 | 电气专业+消防专业+电梯厂家三方联合排查接口 |

## 七、验收配合

### 7.1 验收阶段划分

```
电梯验收流程（三步）：

  第一步：电梯单机验收（特种设备检验机构）
    ↓
  第二步：消防联动验收（消防部门/第三方）
    ↓
  第三步：竣工综合验收（建设单位组织）
```

### 7.2 各专业须配合的验收内容

#### 7.2.1 电梯单机验收（配合特种设备检验机构）

设计院须配合提供的资料：

| 资料 | 提供方 |
|------|-------|
| 电梯井道施工图 | 建筑专业 |
| 结构计算书（机房承重梁、吊钩） | 结构专业 |
| 电梯配电系统图 | 电气专业 |
| 消防电梯前室加压送风验收报告 | 暖通专业 |
| 底坑排水系统验收报告 | 给排水专业 |

检验机构重点检查项（设计相关）：

| 检查项 | 对应设计内容 | 常见不合格原因 |
|-------|------------|--------------|
| 顶部安全空间 | OH值 | 设计时未按最终选型复核 |
| 底坑深度及安全空间 | PD值 | 底坑积水导致实际深度不足 |
| 机房温度 | 通风/空调设计 | 通风量不足或空调未安装 |
| 井道照明 | 电气设计 | 灯位间距>7m或底坑未设灯 |
| 接地电阻 | 电气设计 | 接地线未有效连接 |
| 消防迫降功能 | 消防联动设计 | 接口协议未落实 |
| 轿厢通风 | 暖通设计 | 轿厢通风面积不足（≥轿厢面积1%） |

#### 7.2.2 消防验收（配合消防部门）

电梯相关消防验收要点：

- [ ] 消防电梯从首层到最高层运行时间≤60s
- [ ] 消防电梯载重≥800kg（医院建议≥1000kg）
- [ ] 消防电梯轿厢内设消防专用电话
- [ ] 消防迫降功能正常（触发火灾报警后自动返回首层开门）
- [ ] 非消防电源切断时，消防电梯正常运行
- [ ] 消防电梯前室面积满足规范
- [ ] 前室加压送风风速满足要求（门开启时≥0.7m/s）
- [ ] 消防电梯底坑排水泵运行正常

#### 7.2.3 竣工综合验收

设计院须参与的环节：

| 验收内容 | 设计院角色 |
|---------|----------|
| 电梯运行质量检查 | 确认运行参数与设计一致（速度、载重、开门时间） |
| 无障碍设施验收 | 确认按钮高度、轿厢盲文、语音报站是否到位 |
| 装饰效果验收 | 确认轿厢装饰与整体精装风格协调 |
| 智能系统验收 | 目的层调度、群控策略是否达到设计预期 |
| 使用说明移交 | 确认厂家移交的操作维保手册齐全 |

### 7.3 验收阶段常见问题

| 问题 | 责任方判定 | 处理方式 |
|------|----------|---------|
| 电梯噪音超标 | 若井道/机房尺寸符合设计图→厂家责任；若不符合→施工单位责任 | 厂家调试减振/施工单位整改 |
| 消防迫降不联动 | 电气专业排查接口，厂家排查程序 | 联合调试 |
| 层门与装饰面不齐平 | 若预留尺寸正确→装修施工问题；若预留错误→设计责任 | 按实际情况整改 |
| 底坑积水 | 防水设计是否达标→设计/施工分别排查 | 补做防水+排水泵 |
| 机房温度不达标 | 暖通设计复核风量→不足则增设设备 | 加装排风扇或空调 |

## 八、医疗建筑电梯设计专项注意事项汇总

### 8.1 与其他专项设计的接口

| 专项 | 接口内容 | 负责协调方 |
|------|---------|----------|
| 箱式物流 | 物流井道与电梯井道相邻时的结构处理；物流站点层与电梯停靠层的对应 | 建筑/物流厂家 |
| 气动物流 | 气动管道不得穿越电梯井道 | 暖通/物流厂家 |
| 智能化 | 电梯与门禁、安防、BA系统的联动 | 弱电专业 |
| 精装修 | 电梯厅装饰与公共区域风格统一；层门门套做法 | 精装分包 |
| 标识导视 | 电梯厅楼层指引、轿厢内楼层功能说明 | 标识分包 |
| 净化 | 手术室层电梯厅的洁净度过渡处理 | 净化分包 |

### 8.2 全过程检查清单（Checklist）

**方案阶段：**
- [ ] 电梯配置策划表已完成并经院方确认
- [ ] 竖向流线分析（四分离原则）已落实
- [ ] 井道位置在结构体系中已合理安排
- [ ] 各类型电梯的概算造价已纳入总概算

**初设阶段：**
- [ ] 井道尺寸按"最大包络"原则预留
- [ ] OH值和PD值已校核并满足要求
- [ ] 消防电梯配置满足GB 50016
- [ ] 无障碍电梯配置满足GB 50763
- [ ] 电梯机房/控制柜位置已确定

**施工图阶段：**
- [ ] 厂家深化图已审核
- [ ] 预埋件已转入结构图
- [ ] 电气参数已转入配电系统图
- [ ] 消防联动方案已明确
- [ ] 轿厢装饰方案已确认
- [ ] 各层开门标高已核对

**施工阶段：**
- [ ] 底坑净尺寸实测合格
- [ ] 预埋件位置抽检合格
- [ ] 井道垂直度检测合格
- [ ] 机房承重梁/吊钩安装验收合格
- [ ] 配电送电测试合格

**验收阶段：**
- [ ] 特种设备检验合格
- [ ] 消防联动验收合格
- [ ] 无障碍验收合格
- [ ] 竣工资料齐全移交
