### 3.2 ★卫生学预评价（医疗项目特有）

| 要点 | 说明 |
|------|------|
| 评价内容 | 医疗工艺流程合理性、感控措施、洁污分流、通风系统等 |
| 评价时机 | 初设阶段（方案确定后、施工图前） |
| 评价单位 | 有资质的卫生评价机构 |
| 设计院配合 | 提供平面图、流线分析图、通风方案、给排水方案 |
| ★影响 | 评价意见可能导致平面布局调整，须在施工图前完成 |

### 3.3 ★辐射安全许可（医疗项目特有）

| 要点 | 说明 |
|------|------|
| 适用情况 | 使用放射源（伽马刀、后装）和射线装置（CT/DSA/加速器） |
| 申请时机 | 项目建设前申请，设备安装前取得 |
| 设计院配合 | 提供放射设备清单、机房布局图、防护方案 |
| ★辐射防护预评价 | 由有资质单位编制，是防护设计的依据（详见5.06） |

### 3.4 装配式建筑专项评审

> **★ 原文强调：** "装配式建筑分包单位应主动掌握最新政策，有提醒设计方及甲方的义务和责任，且装配率方案对土建各专业的影响和要求应该在施工图之前明确提出。"

| 要点 | 说明 |
|------|------|
| 政策要求 | 各地装配率要求不同（通常≥30%-50%） |
| 评审时机 | 方案或初设阶段 |
| 对土建的影响 | 预制构件影响结构节点设计、外墙做法、管线预埋 |
| ★注意 | "装配式设计，总承包项目，原则应由设计方来深化设计装配式叠合板等，施工单位在施工前，由PC工厂进行拆分加工，此部分工作不属于设计院负责" |
