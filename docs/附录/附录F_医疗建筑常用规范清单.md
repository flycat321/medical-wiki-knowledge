## 使用说明

本清单汇总医疗建筑EPC设计项目中常用的国家标准、行业标准和相关法规。项目开始前各专业负责人须确认已持有最新版本。标注★的为医疗建筑特有或特别重要的规范。

> **注意：规范持续更新，使用前务必确认为最新有效版本。**

## 十一、地方标准提示

> **★ 以下类别的地方标准可能高于或补充国家标准，项目开始前须逐项确认当地要求：**

| 类别 | 说明 | 确认人 |
|------|------|-------|
| 消防 | 地方消防审查特殊要求 | 建筑+各专业 |
| 绿建/节能 | 地方绿建星级要求、节能计算系数 | 暖通+建筑 |
| 装配式 | 地方装配率/预制率要求 | 结构 |
| 人防 | 地方人防配建面积标准 | 建筑 |
| 无障碍 | 地方无障碍特殊要求 | 建筑 |
| 规划 | 地方控规/修规（容积率/密度/限高/退线等） | 建筑 |
| 抗震 | 地方设防烈度和场地类别 | 结构 |
| 医疗 | 地方医疗建筑专项导则（如有） | 建筑 |
