### 3.1 建筑专业

| 控概方向 | 具体措施 | 造价影响 |
|---------|---------|---------|
| 体型系数 | 减少凹凸、控制体型系数≤0.35 | 体型系数每增0.01，单方造价增50-100元 |
| 层高控制 | 标准层避免超高（普通层3.9-4.2m，手术层4.5-4.8m） | 层高每增0.1m，结构+幕墙造价增≈50元/m² |
| 地下室面积 | 严格控制地下室面积（与地上面积比控制在25-30%） | 地下室单方造价约为地上1.5-2倍 |
| 窗墙比 | 控制窗墙比≤0.50（兼顾节能） | 幕墙/窗造价是墙体的3-5倍 |
| 走廊宽度 | 医院走廊虽需满足规范（≥2.4m），但不宜过度放大 | 每增0.1m宽，全院面积增加可观 |
| 做法标准化 | 屋面/地面/墙面做法尽量标准化，减少特殊做法种类 | 减少施工复杂度和材料种类 |
| ★精装分区 | 严格区分高/中/低装修区域，后勤区不过度装修 | 差异可达5-10倍单方造价 |

### 3.2 结构专业

| 控概方向 | 具体措施 | 造价影响 |
|---------|---------|---------|
| 含钢量 | 控制含钢量指标（框架≤50kg/m²、框剪≤55kg/m²） | 钢筋占结构造价40-50% |
| 含混凝土量 | 控制含混凝土量指标（框架≤0.40m³/m²、框剪≤0.45m³/m²） | — |
| 柱网 | 合理柱网（8.1m×8.1m或8.4m×8.4m为医院常用经济柱网） | 过大柱网增加梁截面和钢筋 |
| 基础形式 | 根据地勘选择最经济基础形式 | 桩基础vs天然地基造价差异巨大 |
| 新技术 | ★各种新技术（阻尼器、减隔震）应在项目建议书/可研中考虑进去，否则后期增加投资会超概 | 减隔震可节省上部结构造价但增加基础造价 |
| 预制率 | 装配式预制率过高会增加造价 | 预制率50%以上时边际成本增加明显 |

### 3.3 水电暖专业

| 专业 | 控概方向 | 具体措施 |
|------|---------|---------|
| 给排水 | 管材选型 | PP-R/PVC-U vs 不锈钢/铜管，造价差3-5倍 |
| 给排水 | 系统简化 | 中水/雨水回收系统须验算经济性（部分项目投入>节水收益）|
| 暖通 | 空调系统 | 多联机vs冷水机组+末端，大面积项目后者更经济 |
| 暖通 | 净化空调 | ★手术室等级配比合理化（不必全做Ⅰ级） |
| 电气 | 变压器容量 | 按需求系数合理计算，避免过度配置 |
| 电气 | 电缆选型 | 合理选择电缆截面和材质（铜vs铝芯） |
| 电气 | 智能化系统 | ★智能化子系统须做取舍，非必须系统可列为预留 |

### 3.4 各专项设计

| 专项 | 控概方向 |
|------|---------|
| 净化 | 手术室等级配比优化（Ⅰ级占比≤20%） |
| 精装修 | 分区控标，后勤区基础装修 |
| 幕墙 | 减少异形造型，增加标准化单元 |
| 物流 | 气动物流站点数量合理化（非每间诊室都需要） |
| 弱电 | 智能化系统做减法，分期实施 |
| 防护 | ★按实际设备参数计算防护厚度，不过度设防 |
| 景观 | 控制硬景/软景比例，乔木规格合理化 |
