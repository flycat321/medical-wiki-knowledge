### 5.1 空调方案（需甲方确认）

| 确认事项 | 选项 | 影响 |
|---------|------|------|
| 冷热源形式 | 冷水机组/多联机/地源热泵等 | 造价差异大 |
| 锅炉房方案 | 燃气/电/蒸汽 | 决定锅炉房位置和管道 |
| 净化空调冷热源 | 独立冷站/共用冷站 | ★须与净化分包协调 |
| 合同能源管理 | 是否采用EMC模式 | 影响设备采购方式 |

### 5.2 与其他专业配合

| 配合专业 | 注意事项 |
|---------|---------|
| 建筑 | ★外墙/外窗留洞提给建筑/幕墙；排烟方案配合开窗；外立面风井管道构筑物沟通 |
| 结构 | 设备荷载（尤其屋面设备）尽早提交；穿梁穿板留洞 |
| 给排水 | 冷却水/锅炉给排水配合 |
| 电气 | 设备配电/消防联动 |
| 净化分包 | ★净化空调与舒适性空调系统划分；新风处理方案；冷热源接口 |

### 5.3 ★医用气体系统

| 气体 | 用途 | 供气方式 |
|------|------|---------|
| 氧气 | 病房/手术室/ICU/急诊 | 液氧站+汇流排+管道 |
| 负压吸引 | 病房/手术室/ICU | 真空泵站+管道 |
| 压缩空气 | 呼吸机/麻醉机驱动 | 空压机站+管道 |
| 笑气(N₂O) | 麻醉 | 汇流排+管道 |
| 氮气 | 手术器械驱动 | 汇流排+管道 |
| 二氧化碳 | 腹腔镜手术 | 钢瓶 |
| 麻醉废气排放 | 手术室 | ★专用排放系统，不得排入回风 |

> **★ 医用气体由设计院暖通/给排水专业设计（各院分工不同），不由净化分包负责。**

### 5.4 错题本

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 防火卷帘1米范围内布置管线 | 防火卷帘无法正常下落 | ★防火卷帘周围按建筑要求留净空 |
| 净化空调与舒适性空调划分不清 | 冷热源接口扯皮 | 初设阶段书面确认划分界面 |
| 屋面设备荷载未尽早提交 | 结构加固 | 方案阶段即初步估算 |
| 风管穿越电梯井道 | ★违反强制条文 | 严禁 |
| 外墙留洞未提给幕墙 | 幕墙封板后凿洞 | ★所有外墙留洞必须提给建筑/幕墙 |
