# 5.16 停车系统与交评专项设计管理细则

---

## 一、概述

医院交通与停车设计直接影响患者就医体验和急救效率。

---

## 二、交通评价（交评）

### 2.1 ★医院交评的特殊性

| 特殊性 | 说明 |
|-------|------|
| ★门诊高峰 | 上午8:00-10:00为就诊高峰，短时间大量集中到达 |
| ★急救通道 | 120急救车专用通道须独立设置，不受社会车辆影响 |
| 人车分流 | 患者步行入口与车行入口严格分离 |
| 出入口数量 | 大型医院建议≥3个车行出入口（门诊/住院/急诊+后勤分开）|

### 2.2 交评报告对设计的约束

| 约束 | 影响 |
|------|------|
| 出入口位置/数量 | 建筑总图调整 |
| 出入口与交叉口距离 | ≥70m（主干道）/≥50m（次干道） |
| 交通组织 | 单行/双行、渠化方案 |
| ★急诊独立出入口 | 必须设置，且有最短路径到城市主要道路 |

## 三、停车系统设计

### 3.1 停车数量计算

| 参考标准 | 数值 | 说明 |
|---------|------|------|
| 按床位 | 0.5-1.0车位/床 | 各地标准不同 |
| 按日门诊量 | 1车位/15-20人次 | — |
| ★充电桩 | ≥总车位15-20% | 逐年提高，建议预留30%管线 |

### 3.2 ★医院停车分区

| 分区 | 说明 |
|------|------|
| 门诊停车区 | 周转率高，建议近门诊出入口 |
| 住院停车区 | 周转率低（日停），可较远 |
| 急诊停车区 | ★不得使用机械车位，须即停即走 |
| 职工停车区 | 地下或较远区域 |
| 后勤/货运区 | 独立通道，不与患者车辆混流 |

### 3.3 充电桩配置

| 类型 | 功率 | 位置 |
|------|------|------|
| 慢充（AC） | 7kW | 职工停车区（日间充满） |
| 快充（DC） | 60-120kW | 门诊区/急诊区旁（短时补电） |
| 预留管线 | — | 总车位30%预留充电管线 |

## 四、停车管理系统

### 4.1 系统组成

| 系统 | 功能 |
|------|------|
| 车牌识别 | 出入口自动识别 |
| 缴费系统 | 扫码/ETC/现金 |
| 车位引导 | 超声/视频检测空位，引导屏显示 |
| 反向寻车 | 输入车牌号查找车辆位置 |
| ★预约停车 | 与门诊预约系统联动，预约患者优先车位 |

### 4.2 ★与门诊系统联动

```
理想流程：

  患者手机预约挂号 → 系统分配停车位 → 到院车牌识别自动放行
  → 引导屏指引至预留车位 → 就诊结束自动扣费 → 出场
```

## 五、常见错误

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 急诊出入口与门诊共用 | 急救车被社会车辆堵塞 | ★急诊独立出入口 |
| 充电桩预留不足 | 后期加装需重新开挖 | 建设期预留30%管线 |
| 门诊区停车位不足 | 高峰期拥堵严重 | 门诊区车位占比≥50% |
| 地下车库高度不够 | 救护车/大型SUV无法进入 | 净高≥2.4m，急救通道≥2.8m |
