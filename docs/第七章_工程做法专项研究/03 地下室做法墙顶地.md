### 3.1 方案阶段要点

| 要点 | 说明 |
|------|------|
| 层高确定 | 停车区≥3.6m（净高≥2.2m）；设备区≥4.2m |
| 防水等级 | 一级防水（医院地下室标准） |
| 柱网 | 停车区柱网与上部功能柱网协调 |
| 出入口 | 车行+人行+设备通道 |

### 3.2 施工图阶段要点

| 部位 | 做法建议 | 造价参考 |
|------|---------|---------|
| 地面（停车区） | 混凝土硬化+环氧地坪漆（标准）/ 金刚砂耐磨（高标准） | 50-100元/m² |
| 地面（设备区） | 混凝土硬化+环氧自流平 | 60-120元/m² |
| 墙面 | 混凝土面+乳胶漆（经济）/ 抹灰+乳胶漆（标准） | 20-50元/m² |
| 顶面 | 结构板底刷白（停车区）/ 吊顶（设备区视需求） | 10-30元/m² |
