### 2.1 项目建议书配合要点

项目建议书通常由院方或咨询公司编制，设计院在此阶段介入程度有限，但应争取参与以下内容：

| 介入内容 | 设计院角色 | 重要性 | 说明 |
|---------|----------|-------|------|
| 建设规模论证 | 提供参考意见 | 高 | 床位数、日门诊量的合理性直接影响后续设计规模 |
| 用地条件初判 | 提供选址建议 | 高 | 地形地貌、周边道路、市政管网条件 |
| 投资估算框架 | 提供造价参考 | 中 | 按床位单价或平米单价估算 |
| 建设周期初估 | 提供设计周期建议 | 中 | EPC模式下设计与施工交叉，总周期短于传统模式 |

> **★ 经验提示：** 项目建议书中的床位数和日门诊量一旦批复，后续调整须重新报批，周期极长。设计院应在此阶段即提醒院方充分论证规模的合理性，避免"批了500床，实际需要800床"的被动局面。

### 2.2 可研报告配合

可行性研究报告是确定项目规模、投资和建设方案的关键文件。EPC项目中设计院应深度参与以下内容：

#### 2.2.1 建设规模论证

| 论证内容 | 数据来源 | 设计院配合工作 |
|---------|---------|--------------|
| 床位数 | 卫健委区域卫生规划、院方发展规划 | 按床位数推算建筑面积（综合医院约110-130m²/床） |
| 日门诊量 | 院方现状数据+增长预测 | 推算门诊面积、候诊空间、停车需求 |
| 科室设置 | 院方学科规划 | ★核对科室清单是否完整（见2.3节） |
| 手术室数量 | 按床位数估算（通常每50-80床配1间手术室） | 影响手术部面积和层高 |
| ICU床位数 | 按总床位5-8%配置 | 影响ICU面积 |

#### 2.2.2 总投资估算

| 费用构成 | 占比参考 | 说明 |
|---------|---------|------|
| 土建安装工程费 | 60-70% | 含结构/机电/装修 |
| 设备及器具购置费 | 15-25% | 医疗设备+信息化+物流等 |
| 工程建设其他费 | 5-10% | 设计费/监理费/前期费等 |
| 预备费 | 5-10% | 不可预见费 |

> **★ 医疗建筑投资参考指标（2024年价格水平）：**

| 医院类型 | 单位造价参考 | 说明 |
|---------|------------|------|
| 三甲综合医院 | 12,000-18,000元/m² | 含土建+安装+精装+专项 |
| 三甲综合医院 | 130-200万/床 | 含全部建安+设备 |
| 专科医院（肿瘤/心血管） | 15,000-22,000元/m² | 大型设备和防护投入高 |
| 康复/养老类 | 8,000-12,000元/m² | — |

#### 2.2.3 建设方案初步确定

| 确定内容 | 深度要求 | 对后续设计的影响 |
|---------|---------|----------------|
| 总体布局方案 | 功能分区、竖向交通、人车分流 | 奠定方案设计基础 |
| 结构形式初判 | 框架/框剪/剪力墙 | 影响概算和工期 |
| 地下室层数 | 1层/2层/3层 | 影响基坑造价和工期 |
| 标准层层高 | 普通层/手术层/设备层 | ★必须在可研阶段确定 |
| 装配式方案 | 装配率要求、预制构件类型 | 影响结构设计和造价 |

### 2.3 ★医疗工艺前策划（核心工作）

**医疗工艺前策划是医疗建筑区别于所有其他建筑类型的核心前置工作。**

#### 2.3.1 科室功能需求调研

项目负责人须组织建筑专业主创，在方案设计开始前完成全院科室需求调研。

**调研方式：**
1. 发放《科室功能需求调研表》→ 各科室主任填写 → 汇总
2. 逐科室座谈（重点科室须现场调研既有院区）
3. 汇总形成《医疗工艺需求书》→ 院方签字确认

**《科室功能需求调研表》应包含以下内容：**

| 调研项 | 说明 | 常见遗漏 |
|-------|------|---------|
| 科室名称及编制 | 门诊/住院/医技各多少人 | — |
| 日工作量 | 日门诊量、日手术量、日检查量 | 院方往往给现状数据，需按发展目标调整 |
| 功能房间清单 | 需要哪些房间、每间面积要求 | ★科室主任的需求通常远超建设标准，须设计院引导合理化 |
| 大型设备清单 | 已有设备+拟采购设备的品牌型号 | ★设备选型直接决定机房条件，遗漏一台CT后期改结构 |
| 特殊流线要求 | 洁污分流、患者隐私、VIP通道等 | — |
| 特殊环境要求 | 洁净度、温湿度、噪声、振动 | 生殖中心VOC要求常被遗漏 |
| 与其他科室关系 | 哪些科室必须相邻/必须同层 | 急诊-影像-手术三角关系是核心 |
| 教学/科研需求 | 示教室、科研实验室 | 教学医院不可忽略 |
| 未来扩展 | 5-10年发展预留 | 院方通常说不清，设计院须引导 |

#### 2.3.2 ★医疗流程确认（五大流线）

在方案设计开始前，须与院方确认以下五大流线的基本组织原则：

```
医疗建筑五大流线：

  ┌─────────────────────────────────────────────────────────┐
  │                                                           │
  │  1. 门诊流线                                              │
  │     患者入口 → 挂号/分诊 → 候诊 → 诊室 → 检查 →          │
  │     取药/缴费 → 离院                                      │
  │     ★ 核心：减少患者走动距离，避免交叉折返                 │
  │                                                           │
  │  2. 急诊流线                                              │
  │     120到达 → 分诊 → 抢救/留观 → 手术/住院/ICU            │
  │     ★ 核心：最短路径到手术室和ICU，生命通道不可中断         │
  │                                                           │
  │  3. 住院流线                                              │
  │     入院 → 护理单元 → 检查(医技) → 手术 → 护理 → 出院     │
  │     ★ 核心：病床可达性，与医技/手术部垂直交通效率           │
  │                                                           │
  │  4. 医技流线                                              │
  │     门诊患者 ↗                                            │
  │              → 检查(影像/检验/功能) → 报告                 │
  │     住院患者 ↗                                            │
  │     ★ 核心：医技是门诊和住院的"交汇点"，位置至关重要       │
  │                                                           │
  │  5. 后勤流线                                              │
  │     物资配送、餐食配送、被服收发、垃圾收运、遗体运送        │
  │     ★ 核心：与患者流线严格分离，污物不走洁净通道            │
  │                                                           │
  └─────────────────────────────────────────────────────────┘
```

#### 2.3.3 ★急诊-影像-手术"生命三角"

```
这是医疗建筑功能布局的第一性原则：

            手术部
           /      \
          /   ≤5min  \
         /    推床距离  \
       急诊 ————————— 影像中心（CT/DSA）
              ≤3min

  ★ 急诊、影像中心（至少急诊CT）、手术部三者之间的推床距离
    不得超过5分钟，理想状态为同层或上下层直通电梯。
  ★ 急诊科须有独立CT（或距放射科CT≤2分钟推床距离）。
```

#### 2.3.4 ★大型医疗设备选型前置清单

以下设备的选型**必须在施工图出图前确定**，否则机房条件无法落实：

| 设备 | 影响范围 | 最迟确定时间 | 后果（如未确定） |
|------|---------|------------|----------------|
| CT（品牌/64排/128排/双源） | 机房面积、荷载、防护、电力 | 初设阶段 | 机房尺寸不够或防护不达标 |
| MRI（1.5T/3.0T） | ★机房面积/荷载/屏蔽/振动/层高 | 方案阶段 | 结构改动代价极大 |
| 直线加速器（品牌/能量） | ★整个放疗科结构体系 | 方案阶段 | 无法补救 |
| DSA（单C/双C/复合手术室） | 机房面积、悬吊轨道、净化+防护 | 初设阶段 | — |
| PET-CT / PET-MR | 机房+候诊区面积、防护、衰变池 | 初设阶段 | — |
| 手术机器人（达芬奇等） | 手术室面积（≥60m²）、吊臂预留 | 方案阶段 | 手术室太小放不下 |
| 洗衣龙 | 洗衣房面积、基础、水电蒸汽 | 施工图阶段 | 影响较小 |
| 中心供应室灭菌器 | CSSD面积、蒸汽、排水 | 初设阶段 | — |
| 箱式物流系统 | 井道预留、层高 | 方案阶段 | 结构已浇筑无法开洞 |

> **★ 经验教训：** 多数项目最大的变更来源就是设备——"院方领导去参观了某医院，回来要加一台3.0T MRI"——如果结构已封顶，这一台MRI可能导致整层楼板加固甚至无法实施。项目负责人必须在方案阶段反复与院方确认设备清单，并书面锁定。
