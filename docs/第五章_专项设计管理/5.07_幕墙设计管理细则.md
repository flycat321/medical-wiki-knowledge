# 5.07 幕墙设计管理细则

---

## 一、概述

医疗建筑幕墙设计需兼顾节能性能、自然通风、排烟功能和医疗功能特殊需求。

---

## 二、各专业内部配合

### 2.1 建筑专业

| 配合事项 | 要求 |
|---------|------|
| 立面设计 | 建筑提供立面方案，幕墙按立面深化 |
| 开启扇 | ★自然通风开启扇位置须建筑统一控制，兼顾立面美观和通风需求 |
| 排烟窗 | ★防排烟方案应尽早与建筑配合立面开窗方案 |
| 遮阳 | 医院病房朝向/手术室遮阳需求须在幕墙方案中体现 |

### 2.2 结构专业

| 配合事项 | 要求 |
|---------|------|
| 预埋件 | 幕墙结构埋件须在主体施工图中标注 |
| 荷载 | 幕墙自重、风荷载传递至主体结构的计算 |
| 层间变位 | 结构提供层间位移角，幕墙按此设计变形缝 |

### 2.3 给排水专业

| 配合事项 | 要求 |
|---------|------|
| 管道避让 | 外墙给排水立管出屋面处须与幕墙协调 |
| 冷凝水 | 幕墙凝结水排放路径 |

### 2.4 电气专业

| 配合事项 | 要求 |
|---------|------|
| 防雷 | ★多高的建筑需要增加防雷接地，须有文件依据 |
| 泛光 | 泛光照明灯具安装位置与幕墙协调 |
| 穿线 | 外墙穿线管位置须与幕墙设计师确认 |

### 2.5 暖通专业

> **★ 以下三条为原文重点强调内容：**

| 配合事项 | 要求 |
|---------|------|
| ★外墙/外窗留洞 | 所有暖通外墙、外窗的留洞都要提给建筑专业和幕墙设计师 |
| ★防排烟 | 防排烟方案应尽早与建筑专业商量，配合立面开窗方案 |
| ★外立面构筑物 | 所有出外立面的风井、管道和构筑物应与建筑专业沟通，避免影响外立面美观 |

---

## 三、分包方协作

幕墙须与以下分包方协调外立面：

| 分包方 | 协调内容 |
|-------|---------|
| 泛光照明 | 灯具安装位置、线缆路由 |
| 精装修 | 室内侧收边收口 |
| 净化专项 | 净化区域外墙不设开启扇 |
| 标识VI | 建筑标识安装面 |

---

## 四、常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| 暖通留洞未提给幕墙 | 幕墙封板后凿洞破坏防水 | 暖通提资必须含外墙留洞清单 |
| 排烟窗位置与幕墙分格冲突 | 排烟面积不够或幕墙分格被迫改动 | 方案阶段即协调 |
| 幕墙预埋件遗漏 | 后期打化学锚栓影响防水 | 预埋件图纸提前审核 |
| 风井出屋面影响美观 | 业主不满意 | 暖通与建筑提前沟通造型处理 |
| 防雷接地未连通 | 验收不通过 | 电气出防雷接地方案幕墙配合 |
