### 3.1 电子版管理

| 要求 | 说明 |
|------|------|
| 命名规则 | `项目代码_专业_图纸编号_版本号`，如 `JN01_JZ_A-01_V3` |
| 版本控制 | 每次修改版本号+1，保留历史版本 |
| 云盘目录 | 按专业/阶段分文件夹，结构与实体资料夹一致 |
| 存储格式 | 设计文件：DWG/DWF；文档：PDF+源文件 |
| ★修改标记 | 修改过的图纸须用云线标注修改范围，图签中标注修改版次和日期 |

### 3.2 纸质版管理

| 要求 | 说明 |
|------|------|
| 出图签章 | 按注册师签章管理规定执行 |
| 蓝图/白图 | 正式交付用蓝图/硫酸纸；内部审核/配合用A3白图 |
| ★规格统一 | 同一次交付的图册必须统一规格（全部A3或全部A1，不得混用） |
| 交付签收 | 每次交付须有签收单，记录交付日期、份数、接收人 |

> **★ 经验教训（济南第一医院）：** 三个专项设计汇报时，景观标识打印的是A3图册，泛光打印的是A4图册，院方书记很生气。**同一次交付的图册项目负责人一定要提前统一规格并审查。**

| 阶段 | 交付物 | 份数 | 格式 | 签收人 |

| 竣工图 | 竣工图（全专业） | | A1 | |
