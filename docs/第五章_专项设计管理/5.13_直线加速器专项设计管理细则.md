# 5.13 直线加速器专项设计管理细则

---

## 一、概述

直线加速器机房是医疗建筑中**单体设计复杂度最高**的空间，涉及超厚混凝土结构、辐射防护迷道、精密空调、大功率供电、大型设备运输等多重挑战。

**★ 核心原则：设备品牌型号必须在结构施工图出图前确定，因为不同品牌的能量等级、机头旋转半径、机房尺寸要求差异巨大，结构一旦浇筑无法修改。**

---

## 二、策划阶段

### 2.1 设备选型对机房的决定性影响

| 参数 | 6MV加速器 | 10MV加速器 | 15MV及以上 | 说明 |
|------|----------|----------|-----------|------|
| 治疗室面积 | ≥50m² | ≥60m² | ≥70m² | 含治疗床旋转空间 |
| 净高 | ≥4.0m | ≥4.5m | ≥4.5m | 机头旋转包络 |
| 主屏障壁厚 | 1.5-1.8m | 1.8-2.2m | 2.2-2.5m | 混凝土 |
| 次屏障壁厚 | 1.0-1.2m | 1.2-1.5m | 1.5-1.8m | — |
| 顶板厚度 | 1.2-1.5m | 1.5-1.8m | 1.8-2.0m | — |
| 迷道长度 | 6-8m | 8-10m | 10-12m | — |
| ★中子防护 | 不需要 | 可能需要 | ★必须 | ≥10MV产生中子辐射 |
| 设备总重 | 3-5吨 | 4-6吨 | 5-8吨 | 含机架 |
| 电力需求 | 100-200kVA | 200-350kVA | 350-500kVA | 专用变压器 |

> **★ 品牌差异举例：** 瓦里安TrueBeam与医科达Versa机头旋转半径差200mm以上，直接影响治疗室最小面积。务必拿到选定品牌的机房条件书再出图。

### 2.2 需院方确认的事项

- [ ] 加速器品牌和型号（★最关键）
- [ ] 能量等级（6MV/10MV/15MV/18MV）
- [ ] 是否配置多叶准直器（MLC）、锥形束CT（CBCT）
- [ ] 未来是否升级预留（如预留质子治疗空间）
- [ ] 放疗科整体规模（几台加速器、是否含伽马刀/后装）

## 三、建筑设计要点

### 3.1 平面布局

```
典型直线加速器机房平面（俯视）：

  ┌─────────────────────────────────┐
  │                                 │
  │     治疗室（≥60m²）              │
  │     净高≥4.5m                    │
  │     壁厚1.5-2.5m混凝土           │
  │                                 │
  │          [治疗床]                │
  │                                 │
  └──────────┐    ┌────────────────┘
             │迷  │
             │道  │ 宽≥1.8m
             │    │ 长6-12m
  ┌──────────┘    └────────────────┐
  │                                 │
  │     控制室/操作间                │
  │     观察窗（铅玻璃）             │
  │                                 │
  └─────────────────────────────────┘

  ★ 迷道入口方向不得正对治疗束方向
  ★ 迷道宽度须能通过治疗床（≥1.8m）
```

### 3.2 ★运输路径

| 要求 | 数值 | 说明 |
|------|------|------|
| 通道宽度 | ≥3.0m | 加速器机架含包装宽度约2.5m |
| 通道净高 | ≥3.0m | — |
| 转弯半径 | ≥4.0m | 需提前模拟 |
| 地面承重 | ≥10kN/m²（运输路径全程） | 含运输工具重量 |
| 台阶 | ★不允许有任何台阶 | 坡道坡度≤1:8 |
| 门洞 | ≥2500×3000mm | 运输通道门 |
| ★吊装口 | 部分情况须预留屋面吊装口 | 设备拆解运入更常见 |

> **★ 建议在施工图中出专项"设备运输路径图"，从建筑物外部到治疗室逐段标注宽度、净高、荷载、转弯半径。**

### 3.3 迷道设计

| 参数 | 要求 |
|------|------|
| 形式 | L型或Z型（不得直通） |
| 宽度 | ≥1.8m（治疗床+操作人员通过） |
| 长度 | 6-12m（视能量等级和入射角度）|
| 壁厚 | ≥1.0m混凝土 |
| 地面 | 平整无障碍（治疗床推行）|
| ★中子防护 | ≥10MV时迷道入口设含硼聚乙烯板 |
| 防护门 | 重达8-15吨，电动/气动，门框须专门设计基础 |

## 四、结构设计要点

| 事项 | 要求 |
|------|------|
| 混凝土等级 | ≥C30，密度≥2350kg/m³ |
| 壁厚 | 1.5-2.5m（主屏障）、1.0-1.5m（次屏障）|
| 顶板 | 1.5-2.0m厚 |
| 底板 | 加厚处理（设备基础+防护）|
| ★大体积混凝土 | 壁厚>1.0m属于大体积混凝土，须防裂措施（分段浇筑、控温、养护）|
| 施工缝 | 严格控制，施工缝处辐射防护须验算 |
| 防护门基础 | 门重8-15吨，门框/地轨须专门设计基础 |
| 吊钩 | 治疗室和迷道顶部预埋检修吊钩 |

> **★ 直线加速器机房结构设计属于特种结构，建议由有经验的结构工程师专项负责。**

## 五、各专业配合

### 5.1 暖通

| 事项 | 要求 |
|------|------|
| 治疗室空调 | 精密空调，温度22-26°C、湿度40-60% |
| 迷道通风 | 送风（微正压），防止放射性气体（≥10MV产生臭氧）|
| 设备冷却 | 冷却水系统（含冷却塔或冷水机组），冷却水量大 |
| ★臭氧排放 | ≥10MV加速器产生臭氧，治疗室须加强通风（≥6次/h）|

### 5.2 电气

| 事项 | 要求 |
|------|------|
| ★专用变压器 | 200-500kVA，电压稳定性≤±2% |
| 供电等级 | 一级负荷 |
| 接地 | 设备接地+辐射防护接地 |
| 辐射警示 | 治疗室门口辐射警示灯+声光报警，与加速器运行联锁 |
| 门禁联锁 | 加速器运行→治疗室门锁→强行开门→加速器停机 |
| 监控 | 治疗室内CCTV，操作间可实时监控患者 |
| 对讲 | 操作间↔治疗室对讲系统 |

### 5.3 给排水

| 事项 | 要求 |
|------|------|
| 冷却水 | 专用管路到设备间 |
| 排水 | 治疗室地面排水（清洁用）|
| 消防 | 治疗室面积大，须确认消防系统（气体灭火/喷淋的适用性）|

## 六、施工/验收

### 6.1 施工特殊要求

- 大体积混凝土分段浇筑方案（每层≤500mm，控制水化热温升≤25°C）
- 养护≥14天（保湿养护）
- ★施工缝处增设止水钢板+外贴防护材料
- 防护门安装须与防护公司配合，门框预埋须在浇筑前完成

### 6.2 验收

| 验收项 | 方法 | 标准 |
|-------|------|------|
| 混凝土强度 | 试块+回弹 | ≥C30 |
| 壁厚 | 实测 | 不得小于设计值 |
| 辐射防护 | 第三方检测 | 见5.06_防辐射细则 |
| 设备安装 | 厂家验收 | 按厂家标准 |
| 联锁功能 | 功能测试 | 门禁+警示灯+紧急停机 |
