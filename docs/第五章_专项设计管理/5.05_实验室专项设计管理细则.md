# 5.05 实验室专项设计管理细则

---

## 一、概述

医疗建筑实验室类型多样，生物安全等级要求严格，与普通实验室设计差异显著。

### 涉及的实验室类型

| 类型 | 生物安全等级 | 主要功能 | 特殊要求 |
|------|------------|---------|---------|
| 临床检验实验室 | BSL-2 | 血液/体液/分泌物检验 | 通风、废液处理 |
| 微生物实验室 | BSL-2 | 细菌培养/药敏 | ★负压、生物安全柜 |
| PCR实验室 | BSL-2 | 基因扩增检测 | ★四区压差梯度 |
| 病理实验室 | BSL-1/2 | 切片/制片/染色 | 通风（甲醛排放）|
| 动物实验室 | BSL-1/2/3 | 动物实验 | ★独立设施、特殊通风 |
| 生物安全P3实验室 | BSL-3 | 高致病性病原体 | ★最严格的负压/密封/消毒 |
| 共享实验室平台 | BSL-2 | 科研用 | 灵活隔断 |

---

## 二、策划阶段

### 2.1 ★生物安全等级对建筑条件的要求

| BSL等级 | 围护结构 | 通风 | 压差 | 废物处理 | 门禁 |
|---------|---------|------|------|---------|------|
| BSL-1 | 普通装修 | 自然/机械通风 | 无强制要求 | 普通处理 | 门锁即可 |
| BSL-2 | 密封（可清洗） | 机械通风 | 微负压（建议） | 高压灭菌 | 门禁+标识 |
| BSL-3 | 气密围护 | ★全新风全排风 | ★负压≥-15Pa | 双扉高压灭菌器 | 门禁+互锁+监控 |

### 2.2 ★PCR实验室四区设置

```
PCR实验室"四区"布局：

  试剂准备区(+5Pa) → 标本制备区(-5Pa) → 扩增区(-10Pa) → 分析区(-15Pa)
       │                    │                   │                │
       └── 正压               └── 负压 ──────────┴── 逐级递减 ──────┘

  ★ 关键要求：
  - 四区之间单向流动，人员物品不得逆向
  - 每区设缓冲间
  - 通风独立，排风不得回流
  - 建筑面积：每区≥15-20m²，四区合计≥80-100m²
```

### 2.3 需院方确认的事项

- [ ] 实验室类型和数量
- [ ] 是否需要P3实验室（审批流程长、造价高）
- [ ] PCR实验室检测通量（影响面积和设备数量）
- [ ] 动物实验室是否设置（需单独审批）
- [ ] 实验室家具品牌偏好

## 三、设计阶段各专业要点

### 3.1 建筑专业

| 要点 | 要求 |
|------|------|
| 模数 | 实验室按3.6m/3.9m模数设计，便于家具和设备排布 |
| 层高 | 净高≥2.8m（吊顶内预留通风管道空间≥800mm）|
| 围护 | BSL-2以上须气密围护，门设观察窗 |
| 缓冲间 | BSL-2设单缓冲，BSL-3设双缓冲（含淋浴）|
| ★气锁 | BSL-3实验室入口设气锁，门互锁 |

### 3.2 暖通专业（核心）

| 要点 | 要求 |
|------|------|
| 微生物室 | ★负压通风，排风100%全排（不回风）|
| 化学通风柜 | 面风速≥0.5m/s，排风直接排出（含过滤/中和处理）|
| 病理实验室 | ★甲醛排风专用系统，排风含活性炭过滤 |
| PCR实验室 | 四区独立通风，压差梯度控制 |
| BSL-3 | ★全新风全排风，HEPA过滤后排放，排风口远离人员 |
| 换气次数 | BSL-2：≥8-10次/h；BSL-3：≥12-15次/h |

### 3.3 给排水专业

| 要点 | 要求 |
|------|------|
| 实验室废水 | ★不得直排市政，须经实验室废水处理系统达标后排放 |
| 酸碱废水 | 中和处理 |
| 纯水系统 | 超纯水/纯水接口到各实验台 |
| 紧急喷淋/洗眼器 | BSL-2以上实验室须设，距操作台≤15m |

### 3.4 电气专业

| 要点 | 要求 |
|------|------|
| 不间断电源 | 精密仪器/冰箱/生物安全柜须配UPS |
| 接地 | 精密仪器单独接地，接地电阻≤1Ω |
| 照明 | 一般≥300lx，精密操作区≥500lx |

| 范围 | 设计院 | 实验室分包 |
|------|-------|----------|
| 建筑平面布局 | ✓ 主导 | 配合优化 |

| 纯水系统 | 给排水主管到位 | ✓ 负责末端分配 |
| 气体管路 | — | ✓ |
| 废水处理 | 给排水总管设计 | ✓ 负责处理设备 |

## 五、常见错误（错题本）

| 问题 | 后果 | 预防措施 |
|------|------|---------|
| PCR四区未设独立通风 | 交叉污染导致检测结果不准 | 每区独立风管，不得合并 |
| 微生物室未做负压 | 气溶胶逸出风险 | 必须负压+生物安全柜 |
| 实验室废水直排 | 环保违法 | 必须经废水处理达标后排放 |
| 病理甲醛通风不足 | 职业健康风险 | 甲醛操作区域局部排风+全室排风 |
