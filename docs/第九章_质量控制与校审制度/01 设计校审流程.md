### 1.1 四级校审制度

```
设计文件校审流程（四级）：

  设计人（自校）→ 校对人（互校）→ 审核人（专业审核）→ 审定人（综合审定）
      │                │                  │                    │
   设计完成后       同专业同事         专业负责人           项目负责人/
   逐项自查         交叉检查           技术把关            总工程师

  ★ 各级校审须在图纸上签字（图签栏），未完成校审不得出图
```

### 1.2 各级校审职责

| 级别 | 校审人 | 校审内容 | 时间要求 |
|------|-------|---------|---------|
| 自校 | 设计人本人 | 尺寸/标注/计算/做法/文字 | 出图前必须完成 |
| 互校 | 同专业同事 | 与设计人交叉检查，重点查遗漏和错误 | 1-2个工作日 |
| 审核 | 专业负责人 | 技术方案合理性、规范符合性、与其他专业一致性 | 2-3个工作日 |
| 审定 | 项目负责人/总工 | 整体品质、跨专业协调、重大技术决策 | 1-2个工作日 |

### 1.3 ★校审不可跳过的红线

| 红线 | 说明 |
|------|------|
| 不得跳级 | 未经自校/互校不得直接送审核 |
| 不得代签 | 校审签字必须本人签署 |
| 不得出图前补签 | 必须先签字再出图 |
| 审核意见必须闭合 | 校审提出的问题必须修改后复核确认 |
